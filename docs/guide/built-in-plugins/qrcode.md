# tinker-qrcode

A QR code generator and scanner plugin for [TINKER](https://github.com/liriliri/tinker), supporting customizable QR code creation and image-based QR code scanning.

![Screenshot](/qrcode.png)

## Features

- **QR code generation** — generate QR codes from any text or URL in real time
- **Customizable size** — choose from preset sizes (300–600) or enter a custom size
- **Color customization** — set foreground and background colors for the QR code
- **Error correction levels** — choose from L, M, Q, or H error correction levels
- **Custom icon** — add a center icon/logo to the QR code
- **QR code scanning** — scan QR codes from image files
- **Copy to clipboard** — copy the generated QR code image to clipboard
- **Save as image** — save the QR code as an image file

## Usage

1. Enter text or a URL in the input area on the left
2. The QR code is generated automatically in real time on the right
3. Adjust **size**, **foreground color**, **background color**, and **error correction** level as needed
4. Click **Set Icon** to add a custom logo to the center of the QR code
5. Click **Copy** to copy the QR code to clipboard, or **Save** to export as an image file
6. Click **Scan QR Code** and select an image to decode a QR code from it
