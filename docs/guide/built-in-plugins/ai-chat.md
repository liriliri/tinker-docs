# tinker-ai-chat

An AI chat plugin for [TINKER](https://github.com/liriliri/tinker) that provides a multi-session conversational interface with support for various AI providers and web search capabilities.

![Screenshot](/ai_chat.png)

## Features

- **Multi-session management** — create, switch, and delete multiple chat sessions
- **Multi-provider support** — works with various AI providers and models
- **System prompts** — customize AI behavior per session with custom system prompts
- **Web search** — AI can search the web and display results inline in the conversation
- **Message management** — retry, delete, or clear messages within a session
- **Session persistence** — conversations are saved and restored across sessions
- **Bilingual UI** — supports English and Chinese

## Usage

1. Select an AI provider and model from the dropdown
2. Click the new chat button to start a conversation
3. Type your message in the input area and press Enter or click the send button
4. The AI will respond, optionally using web search to find relevant information
5. Click the settings icon to configure a system prompt for the current session
6. Use the sidebar to switch between or manage multiple sessions
