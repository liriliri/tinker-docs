# tinker-voice-recorder

A voice recorder plugin for [TINKER](https://github.com/liriliri/tinker), allowing you to record audio from your microphone with waveform visualization and playback preview.

![Screenshot](/voice_recorder.png)

## Features

- **Audio recording** — record audio from your microphone with one click
- **Waveform visualization** — real-time waveform display during recording
- **Pause/resume** — pause and resume recording without losing progress
- **Recording timer** — displays the current recording duration in real time
- **Preview playback** — play back the recorded audio before saving with waveform display
- **Save recording** — save the recording to a file with a custom file name
- **Download** — download the recorded audio file
- **Delete recording** — discard the current recording and start over

## Usage

1. Click the **Record** button to start recording from your microphone
2. Grant microphone access when prompted by the browser
3. Use **Pause** and **Resume** to control the recording as needed
4. Click **Stop** to finish recording
5. Preview the recording with the built-in playback controls
6. Click **Save** to save the recording to a file, or **Delete** to discard it
