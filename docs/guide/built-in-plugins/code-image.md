# tinker-code-image

A code image plugin for [TINKER](https://github.com/liriliri/tinker) that generates beautiful, shareable images from code snippets with syntax highlighting and customizable themes.

![Screenshot](/code_image.png)

## Features

- **Syntax highlighting** — powered by Shiki with support for multiple programming languages
- **Theme selection** — choose from various color themes for light and dark modes
- **Line numbers** — toggle line number display
- **Custom filename** — edit the filename shown in the macOS-style title bar
- **Export options** — copy the image to clipboard as PNG or save it as a file
- **Bilingual UI** — supports English and Chinese

## Usage

1. Type or paste code into the editor
2. Select the programming language from the dropdown
3. Choose a color theme and toggle dark/light mode as desired
4. Optionally enable line numbers or edit the filename
5. Click the copy button to copy the image to the clipboard, or the save button to download it as a PNG file
