# tinker-password-manager

A password manager plugin for [TINKER](https://github.com/liriliri/tinker), powered by the KeePass (KDBX) format for secure password storage and management.

![Screenshot](/password_manager.png)

## Features

- **KeePass compatible** — create, open, and save KDBX database files compatible with KeePass
- **Group management** — organize entries into hierarchical groups with create, rename, and delete operations
- **Entry management** — store title, username, password, URL, notes, and custom fields for each entry
- **Search** — quickly search across all entries by title, username, URL, or notes
- **Password visibility** — toggle password display with show/hide
- **Copy to clipboard** — one-click copy username or password
- **Key file support** — open databases using password with optional key file
- **Recent files** — quick access to recently opened database files
- **Database lock** — lock the database to protect data when stepping away

## Usage

1. Click **New Database** to create a new password database, or **Open Database** to load an existing KDBX file
2. Enter the **master password** (and optional key file) to unlock the database
3. Use the **group tree** on the left to organize entries into categories
4. Click **Create Entry** to add a new password entry, filling in title, username, password, URL, and notes
5. Select an entry to view or edit its details in the right panel
6. Use the **Search** bar to quickly find entries across all groups
7. Click **Save** to persist changes to the database file
