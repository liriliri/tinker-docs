# tinker-clock

A clock plugin for [TINKER](https://github.com/liriliri/tinker) that displays the current time in three visual styles: flip clock, analog clock, and digital clock, with timezone support.

![Screenshot](/clock.png)

## Features

- **Flip clock** — classic flip animation style display
- **Analog clock** — glass morphism design with smooth hand animations
- **Digital clock** — seven-segment LED-style display with SVG animations
- **Timezone support** — switch between local time and different timezones
- **Fullscreen mode** — display the clock in fullscreen
- **Bilingual UI** — supports English and Chinese

## Usage

1. Select a clock style from the toolbar (flip, analog, or digital)
2. Use the timezone selector to change the displayed timezone
3. Click the fullscreen button to enter fullscreen mode
