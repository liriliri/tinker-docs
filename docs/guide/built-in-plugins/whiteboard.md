# tinker-whiteboard

A whiteboard plugin for [TINKER](https://github.com/liriliri/tinker), powered by [tldraw](https://tldraw.com/) for freeform drawing and diagramming.

![Screenshot](/whiteboard.png)

## Features

- **Freeform drawing** — draw, sketch, and annotate freely on an infinite canvas
- **Shape tools** — create rectangles, ellipses, arrows, lines, and more
- **Text** — add text labels anywhere on the canvas
- **Sticky notes** — add sticky notes for quick annotations
- **Pen & eraser** — draw with the pen tool and erase with the eraser
- **Selection & transformation** — select, move, resize, and rotate objects
- **Undo/redo** — full undo and redo support
- **Export** — export your whiteboard as an image

## Usage

1. Open the plugin to start with a blank infinite canvas
2. Use the toolbar to select drawing tools: pen, shapes, text, sticky notes, etc.
3. Draw and arrange elements on the canvas
4. Use undo/redo to correct mistakes
