# tinker-system-info

A system information viewer plugin for [TINKER](https://github.com/liriliri/tinker), displaying detailed hardware and software information about your computer.

![Screenshot](/system_info.png)

## Features

- **System overview** — view model, manufacturer, and serial number of your system
- **OS information** — display operating system name, version, and release date
- **Processor details** — show CPU chip name, cores, threads, and clock speed
- **Memory information** — view total, used, and available RAM with memory type
- **Graphics & display** — list graphics cards and display resolutions
- **Storage** — show disk layout with size, type, and vendor information
- **Audio devices** — list available audio devices
- **Network interfaces** — display network adapters with MAC address, IPv4, and IPv6 addresses
- **Click to copy** — click on any value to copy it to the clipboard

## Usage

1. Open the plugin to automatically load and display all system information
2. Browse through the sections: **Overview**, **Processor**, **Memory**, **Graphics**, **Display**, **Storage**, **Audio**, and **Network**
3. Click on any value to copy it to the clipboard
