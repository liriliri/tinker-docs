# tinker-mind-map

A mind map plugin for [TINKER](https://github.com/liriliri/tinker), allowing you to create, edit, and organize mind maps with multiple layout options.

![Screenshot](/mind_map.png)

## Features

- **Multiple layouts** — switch between Logical Structure, Mind Map, Organization Structure, and Catalog Organization
- **Outline sidebar** — view and navigate nodes in a tree-structured outline panel
- **Node operations** — insert child/sibling nodes, move, copy, cut, paste, and delete nodes via context menu
- **File operations** — create new mind maps, open existing files (JSON/SMM), and save changes
- **Undo/Redo** — full undo and redo support for all editing actions
- **Zoom controls** — zoom in/out with preset values, or fit the entire map to the canvas
- **Export image** — export the mind map as a PNG image
- **Keyboard navigation** — navigate and edit nodes using keyboard shortcuts
- **Hyperlinks and notes** — attach hyperlinks and notes to nodes

## Usage

1. Launch the plugin to see a default mind map with a root node and sample branches
2. Double-click a node to edit its text
3. Right-click a node to access the context menu for inserting, moving, copying, or deleting nodes
4. Use the **layout selector** in the toolbar to switch between different layout styles
5. Toggle the **sidebar** to view the outline and quickly navigate to any node
6. Click **Save** to save your mind map to a file, or **Open** to load an existing one
7. Use the zoom controls in the bottom-right corner to adjust the view, or click **Fit** to fit the map to the canvas
8. Click **Export Image** to download the mind map as a PNG file
