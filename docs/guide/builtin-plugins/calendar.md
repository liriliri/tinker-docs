# tinker-calendar

A calendar plugin for [TINKER](https://github.com/liriliri/tinker) that provides a full-featured calendar with event management, multiple views, and holiday display including lunar calendar support.

![Screenshot](/calendar.png)

## Features

- **Multiple views** — switch between day, week, and month views
- **Event management** — create, edit, and delete calendar events with support for all-day and timed events
- **Drag and drop** — move events by dragging them to a new date or time
- **Holiday display** — shows holidays for both Western and Chinese calendars
- **Lunar calendar** — displays lunar dates in the month view
- **Sidebar** — view a list of events for the selected date
- **Bilingual UI** — supports English and Chinese

## Usage

1. Use the toolbar to switch between day, week, and month views
2. Click the add button or double-click a date to create a new event
3. Fill in the event title, start/end date and time, and optionally mark it as all-day
4. Click an existing event to edit or delete it
5. Toggle the sidebar to view all events for the selected date
6. Use the previous/next buttons or click "Today" to navigate between dates
