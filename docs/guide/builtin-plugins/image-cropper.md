# tinker-image-cropper

An image cropping plugin for [TINKER](https://github.com/liriliri/tinker), with rotation, flipping, resizing, and aspect ratio controls.

![Screenshot](/image_cropper.png)

## Features

- **Interactive cropping** — drag to select a crop area with real-time preview
- **Aspect ratio presets** — Free, Original, 1:1, 4:3, 3:2, 16:9
- **Custom crop size** — set exact crop dimensions in pixels
- **Rotate** — rotate the image left or right by 90 degrees
- **Flip** — flip the image horizontally or vertically
- **Resize** — resize the image to specific dimensions with optional aspect ratio lock
- **Screen capture** — capture the screen directly and crop
- **Clipboard support** — paste images from clipboard, copy cropped result
- **Undo/Redo** — full history support for crop operations
- **Save options** — overwrite the original file or save as a new file

## Usage

1. Click **Open** to load an image, capture the screen, or paste from clipboard
2. Drag on the image to define the crop area
3. Optionally select an **Aspect Ratio** preset or set a custom crop size
4. Use rotate, flip, or resize controls as needed
5. Click **Crop** to apply the crop
6. Click **Copy** to copy the result to clipboard, or **Save** to save the cropped image
