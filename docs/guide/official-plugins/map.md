# tinker-map

A map plugin for [TINKER](https://tinker.liriliri.io/), providing an interactive map interface with location browsing and search.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-map/screenshot.png)

## Features

- **Interactive Map** — Explore maps powered by Leaflet
- **Location Search** — Search locations by name or description
- **Location List** — Sidebar with all locations, click to focus on the map
- **Dark Mode** — Supports dark theme
- **i18n** — Supports English and Chinese

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-map
```

## Usage

1. Browse locations on the map or in the sidebar list
2. Search for locations using keywords
3. Click a location to focus the map on it
