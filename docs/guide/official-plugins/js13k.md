# tinker-js13k

A JS13K games plugin for [TINKER](https://github.com/liriliri/tinker), featuring a curated collection of JS13K competition entries playable directly in the app.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-js13k/screenshot.png)

## Features

- **16 Curated Games** from JS13K competitions (2017–2025)
- **Search** games by name, author, or description
- **Responsive Grid** layout adapting to window size
- **In-app Playback** with fullscreen support
- **Dark Mode** support

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-js13k
```

## Usage

1. Browse the game collection in the grid view
2. Search for games using the search bar
3. Click a game card to start playing
4. Click the back button to return to the game list
