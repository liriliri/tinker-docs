# tinker-translate

Translation plugin with Google Translate, Bing Translate, and AI translation support.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-translate/screenshot.png)

## Features

- **Multiple Engines** — Google Translate, Bing Translate, and AI translation
- **Auto Detection** — Automatically detect source language
- **Swap Languages** — One-click swap between source and target languages
- **Paste to Translate** — Quickly paste clipboard content for translation
- **Copy Result** — One-click copy of translation result
- **Bilingual UI** — Supports Chinese and English
- **Dark Mode** — Supports dark theme

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-translate
```

## Usage

1. Select a translation engine (Google, Bing, or AI) from the dropdown
2. Choose source and target languages, or use auto-detection
3. Enter or paste text in the left panel
4. Click **Translate** to start translation
5. View the result in the right panel
6. Click copy to copy the translation result
