# tinker-trending

Trending hot search aggregator plugin, bringing together popular topics from multiple platforms at a glance.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-trending/screenshot.png)

## Features

- **Multi-platform** — Hacker News, GitHub, Zhihu, Weibo, V2EX, Bilibili, Douban, Juejin, Toutiao, Sspai, Tieba, Steam
- **Custom Sources** — Freely add or remove platforms
- **Drag to Reorder** — Drag cards to rearrange display order
- **Refresh All** — Batch refresh data from all platforms
- **Bilingual UI** — Supports Chinese and English
- **Dark Mode** — Supports dark theme

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-trending
```

## Usage

1. Trending data loads automatically when the plugin opens
2. Click an item to open the original link
3. Click the **+** button to add or remove platforms
4. Drag cards to rearrange the display order
5. Click the refresh button to update all platforms
