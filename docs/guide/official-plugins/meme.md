# tinker-meme

A meme search plugin for [TINKER](https://github.com/liriliri/tinker), supporting meme search, browsing, and clipboard copy.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-meme/screenshot.png)

## Features

- **Meme search** by keyword
- **Hot memes** when the search box is empty
- **Infinite scroll** to load more results
- **Click to copy** image to clipboard
- **Right-click to save** image to local disk
- **Dark mode** support
- **i18n** support for English and Chinese

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-meme
```

## Usage

1. Enter keywords in the search box and press Enter
2. Browse memes in the grid; scroll down to load more
3. Click a meme to copy it to the clipboard
4. Right-click a meme and choose "Save" to save it locally

## Data Source

Memes are sourced from [搜狗图片](https://pic.sogou.com/).
