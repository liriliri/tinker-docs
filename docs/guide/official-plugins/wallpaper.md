# tinker-wallpaper

Wallpaper plugin for searching and browsing HD wallpapers, with preview, download, and desktop wallpaper setting support.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-wallpaper/screenshot.png)

## Features

- **Wallpaper Search** — Search HD wallpapers by keyword
- **Wallpaper Browsing** — Grid display with infinite scroll loading
- **Wallpaper Preview** — View full-size images with multiple resolution options
- **Set as Wallpaper** — Set as desktop wallpaper with automatic best-fit resolution
- **Save Wallpaper** — Download wallpapers to local storage
- **Bilingual UI** — Supports Chinese and English
- **Dark Mode** — Supports dark theme

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-wallpaper
```

## Usage

1. Enter keywords in the search bar to find wallpapers
2. Browse the wallpaper grid, scroll to load more
3. Click a wallpaper to open the preview
4. Select the desired resolution
5. Click the set button to apply as desktop wallpaper, or click save to download locally
