# tinker-weather

A weather plugin for [TINKER](https://tinker.liriliri.io/), providing real-time weather information and forecasts.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-weather/screenshot.png)

## Features

- **Real-time Weather** — View temperature, humidity, wind, and UV index
- **3-Day Forecast** — Temperature range bars for upcoming days
- **City Search** — Search cities with pinyin support for Chinese input
- **Recent Cities** — Quickly switch between recently viewed cities
- **Dark Mode** — Supports dark theme

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-weather
```

## Usage

1. Search for a city in the sidebar
2. View current weather and 3-day forecast
3. Switch between recent cities
