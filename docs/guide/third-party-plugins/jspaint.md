# tinker-jspaint

JSPaint plugin for [TINKER](https://github.com/liriliri/tinker).

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-jspaint/screenshot.png)

## About

This plugin integrates [JS Paint](https://github.com/1j01/jspaint), a pixel-perfect web-based MS Paint remake, into TINKER. JS Paint faithfully recreates the classic Windows Paint experience with modern improvements like unlimited undo, transparent image editing, and touch support.

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-jspaint
```
