# tinker-online3dviewer

Online 3D Viewer plugin for [TINKER](https://github.com/liriliri/tinker).

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-online3dviewer/screenshot.png)

## About

This plugin integrates [Online 3D Viewer](https://github.com/kovacsv/Online3DViewer), a free and open-source online 3D model viewer that supports a wide range of file formats including OBJ, STL, GLTF, GLB, 3DM, OFF, PLY, BIM, and more.

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-online3dviewer
```
