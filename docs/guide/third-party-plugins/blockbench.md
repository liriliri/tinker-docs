# tinker-blockbench

Blockbench plugin for [TINKER](https://github.com/liriliri/tinker).

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-blockbench/screenshot.png)

## About

This plugin integrates [Blockbench](https://github.com/JannisX11/blockbench), a free modern model editor for low-poly and boxy models with pixel art textures. Blockbench is primarily used for Minecraft and entity modeling, but it also supports general 3D modeling and animation with features for texturing, animating, and exporting to various formats.

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-blockbench
```
