# tinker-audiomass

AudioMass plugin for [TINKER](https://github.com/liriliri/tinker).

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-audiomass/screenshot.png)

## About

This plugin integrates [AudioMass](https://github.com/pkalogiros/AudioMass), a free full-featured web-based audio and waveform editing tool. AudioMass provides a comprehensive set of audio editing capabilities including cutting, trimming, effects, filters, and format conversion, making it perfect for audio editing tasks directly in your browser.

## Installation

Download and install [TINKER](https://tinker.liriliri.io/), then run:

```bash
npm i -g tinker-audiomass
```
