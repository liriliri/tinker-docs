# tinker-duplicate-cleaner

重复文件清理插件，扫描目录查找并删除重复文件，帮助释放磁盘空间。

![截图](/duplicate_cleaner.png)

## 功能特性

- **重复检测** — 扫描目录，使用 MD5 哈希按内容分组重复文件
- **基于大小的预筛选** — 先按文件大小分组，再通过哈希验证，提升扫描效率
- **文件类型过滤** — 按类别筛选结果：音频、视频、文档、图片或其他
- **交互式选择** — 可展开的重复文件组，使用复选框选择要删除的文件
- **扫描进度** — 扫描过程中实时显示文件数量和总大小
- **可取消扫描** — 随时停止正在进行的扫描
- **安全删除** — 删除前弹出确认对话框，并反馈已删除数量和错误信息
- **选择摘要** — 实时显示已选文件数量和可释放的总大小

## 使用方法

1. 点击文件夹图标或拖放目录开始扫描
2. 等待扫描完成，进度会实时显示
3. 在可展开的结果列表中查看重复文件组
4. 使用工具栏标签按文件类型筛选（音频、视频、文档、图片、其他）
5. 勾选每组重复文件中要删除的文件
6. 点击"清理"按钮并确认，删除选中的重复文件
