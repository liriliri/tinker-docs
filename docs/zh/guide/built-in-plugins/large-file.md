# tinker-large-file

大文件清理插件，扫描目录查找大文件（10 MB 以上），帮助你删除它们以释放磁盘空间。

![截图](/large_file.png)

## 功能特性

- **文件夹扫描** — 选择任意文件夹或拖拽文件夹进行递归扫描
- **最小体积阈值** — 自动识别 10 MB 及以上的文件
- **文件类型筛选** — 按类别筛选结果：音频、视频、文档、图片或其他
- **结果数据表格** — 以清晰的表格展示文件名、路径和大小
- **批量选择** — 使用复选框选择多个文件进行批量删除
- **选择汇总** — 实时显示已选文件数量和总大小
- **删除确认** — 清理前需确认，并提供成功/失败反馈
- **在文件夹中显示** — 点击文件路径可在系统文件管理器中定位
- **扫描进度** — 扫描过程中实时显示已扫描文件数量和总大小
- **取消扫描** — 可随时中止正在进行的扫描

## 使用方法

1. 点击"打开文件夹"按钮或拖拽文件夹到窗口中开始扫描
2. 等待扫描完成，进度会实时显示
3. 浏览结果列表，显示所有 10 MB 及以上的文件，按大小排序
4. 使用筛选标签（全部、音频、视频、文档、图片、其他）缩小范围
5. 使用复选框勾选要删除的文件
6. 点击"清理"按钮，在弹出的对话框中确认删除
