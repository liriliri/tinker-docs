# tinker-media-compressor

音视频压缩插件，基于 FFmpeg，支持批量视频和音频压缩。

![截图](/media_compressor.png)

## 功能特性

- **视频压缩** — 压缩 MP4、MKV、AVI、MOV 和 WebM 文件
- **音频压缩** — 压缩 MP3、M4A、AAC、OGG、FLAC 和 WAV 文件
- **多种压缩模式** — 视频支持 CRF（质量）、比特率、分辨率和目标大小；音频支持比特率、采样率和目标大小
- **质量预设** — 从极低到极佳五个质量级别
- **GPU 加速** — 自动检测 GPU 编码器，加速视频压缩
- **批量处理** — 一次压缩多个文件，显示进度跟踪
- **目标大小** — 压缩到指定文件大小（KB、MB 或 GB）
- **自定义输出目录** — 将压缩文件保存到指定目录或与原文件同目录
- **媒体信息显示** — 查看每个文件的分辨率、帧率、时长、比特率和采样率
- **取消支持** — 随时取消压缩任务

## 使用方法

1. 在工具栏中切换**视频**和**音频**模式
2. 点击**打开**或拖拽媒体文件添加
3. 选择压缩模式并调整质量级别
4. 可选设置输出目录
5. 点击**压缩**开始批量压缩
6. 监控每个文件的进度；点击**取消**可随时停止
7. 压缩后的文件保存到输出目录或与原文件同目录
