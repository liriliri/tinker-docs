# tinker-pomodoro

[TINKER](https://github.com/liriliri/tinker) 的番茄钟插件，通过定时的工作和休息间隔帮助你保持专注。

![Screenshot](/pomodoro.png)

## 功能特性

- **番茄工作法** — 自动循环专注时段、短休息和长休息
- **环形进度条** — 可视化的环形进度指示器显示已用时间
- **轮次追踪** — 追踪当前轮次和已完成的专注次数
- **可配置时长** — 自定义专注时间（默认 25 分钟）、短休息（5 分钟）和长休息（15 分钟）
- **声音提醒** — 计时完成时播放音频提示，支持调节音量
- **跳过和重置** — 跳过到下一阶段或重置整个会话
- **后台计时** — 使用 Web Worker 运行计时器，即使窗口在后台也能精确计时

## 使用方法

1. 启动插件后会显示设定为 25 分钟的专注计时器
2. 点击**播放**按钮开始计时
3. 专注于你的任务，直到计时结束并播放声音提醒
4. 计时器会自动切换到**短休息**（每 4 轮后切换为**长休息**）
5. 点击**跳过**可跳转到下一阶段，或点击**重置**重新开始
6. 调节音量来控制提示音大小
