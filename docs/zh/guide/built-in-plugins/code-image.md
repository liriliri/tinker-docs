# tinker-code-image

代码图片插件，将代码片段生成为美观、可分享的图片，支持语法高亮和自定义主题。

![截图](/code_image.png)

## 功能特性

- **语法高亮** — 基于 Shiki，支持多种编程语言
- **主题选择** — 提供多种配色主题，支持明暗模式
- **行号显示** — 可切换行号显示
- **自定义文件名** — 可编辑 macOS 风格标题栏中显示的文件名
- **导出选项** — 将图片复制到剪贴板（PNG 格式）或保存为文件
- **双语界面** — 支持中文和英文

## 使用方法

1. 在编辑器中输入或粘贴代码
2. 从下拉菜单中选择编程语言
3. 选择配色主题，按需切换明暗模式
4. 可选开启行号显示或编辑文件名
5. 点击复制按钮将图片复制到剪贴板，或点击保存按钮下载为 PNG 文件
