# tinker-config-editor

配置文件编辑插件，提供可视化编辑器，用于浏览和编辑系统及应用程序配置文件。

![截图](/config_editor.png)

## 功能特性

- **自动检测配置文件** — 自动扫描常见配置文件，如 `.npmrc`、`.gitconfig`、`.bashrc`、`.zshrc`、`hosts`、`nginx.conf` 等
- **Monaco 代码编辑器** — 支持 JSON、INI、shell 和纯文本格式的语法高亮编辑
- **撤销/重做和搜索** — 完整的编辑功能，包括撤销、重做和编辑器内搜索（Ctrl+F）
- **保存更改** — 直接从编辑器将修改写回磁盘
- **未保存更改追踪** — 文件修改但未保存时显示可视化提示
- **在文件夹中显示** — 在系统文件管理器中定位配置文件
- **复制内容** — 将整个文件内容复制到剪贴板
- **可调字体大小** — 编辑器字体大小可在 10px 到 32px 之间调整
- **可折叠侧边栏** — 文件列表侧边栏可切换显示，腾出更多编辑空间
- **状态持久化** — 上次打开的文件、字体大小和侧边栏状态跨会话保留

## 使用方法

1. 打开插件，在侧边栏中查看检测到的配置文件列表
2. 点击文件，在 Monaco 代码编辑器中加载
3. 使用语法高亮和行号编辑文件内容
4. 点击保存按钮将更改写回磁盘
5. 使用文件夹图标在系统文件管理器中定位文件
6. 通过设置调整字体大小，自定义编辑器显示效果
