# tinker-image-annotator

图片标注插件，提供绘图、形状、文本和标记等图片标注工具。

![截图](/image_annotator.png)

## 功能特性

- **绘图工具** — 选择、移动、自由画笔、形状（矩形、椭圆、线条、箭头）、文本、放大镜和马赛克
- **形状标注** — 绘制矩形、椭圆、线条和箭头，可配置线条粗细
- **文本标注** — 添加可编辑文本，可调整字体大小
- **马赛克工具** — 对图片区域进行像素化处理，遮挡敏感信息
- **放大镜工具** — 放大局部区域以突出细节
- **图片叠加** — 在画布上插入额外的图片
- **屏幕截图** — 直接截取屏幕并进行标注
- **剪贴板支持** — 从剪贴板粘贴图片，复制标注结果
- **水印** — 添加可自定义文本和颜色的水印
- **颜色选择器** — 前景色和背景色选择，支持互换
- **撤销/重做** — 完整的撤销和重做支持
- **缩放控制** — 放大、缩小和适应视图
- **导出** — 将标注后的图片保存为 PNG、JPG 或 WebP 格式

## 使用方法

1. 点击**打开**加载图片，或截取屏幕，或从剪贴板粘贴
2. 从侧边工具栏选择绘图工具（画笔、形状、文本、放大镜、马赛克等）
3. 在图片画布上绘制标注
4. 根据需要在顶部工具栏中调整线条粗细、字体大小或形状类型
5. 可选开启**水印**并自定义文本和颜色
6. 点击**复制**将结果复制到剪贴板，或点击**保存**下载标注后的图片
