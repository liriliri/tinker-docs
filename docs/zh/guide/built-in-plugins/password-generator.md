# tinker-password-generator

[TINKER](https://github.com/liriliri/tinker) 的确定性密码生成插件，通过主密码和服务名称组合生成唯一密码。

![Screenshot](/password_generator.png)

## 功能特性

- **确定性生成** — 相同的主密码和服务名称始终生成相同的密码，无需存储
- **字符类型控制** — 将每种字符类型（小写字母、大写字母、数字、符号、短横线/下划线、空格）设置为必需、允许或禁止
- **密码长度** — 配置所需的密码长度
- **最大重复次数** — 限制字符的最大重复次数
- **主密码可见性** — 支持显示/隐藏主密码切换
- **复制到剪贴板** — 一键复制生成的密码

## 使用方法

1. 输入**主密码**（你的主密钥短语）
2. 输入**服务名称**（例如 "twitter"、"gmail"）
3. 根据需要调整**密码长度**和**最大重复次数**
4. 通过在**必需**、**允许**或**禁止**之间切换来配置各字符类型
5. 密码会根据输入自动生成
6. 点击**复制**按钮将生成的密码复制到剪贴板
