# tinker-file-shredder

文件粉碎插件，通过多次覆盖写入数据来安全删除文件，防止被恢复。

![截图](/file_shredder.png)

## 功能特性

- **多种粉碎方式** — 可选快速（1 次覆盖）、标准（3 次覆盖，DoD）或深度（7 次覆盖）
- **批量粉碎** — 添加多个文件或整个文件夹进行批量处理
- **拖放支持** — 直接将文件拖放到界面上
- **进度追踪** — 实时显示每个文件的粉碎进度和总体进度
- **支持取消** — 粉碎过程中可随时取消
- **在文件夹中显示** — 粉碎前可在系统文件管理器中定位文件
- **文件详情** — 查看每个已添加文件的名称、路径和大小

## 使用方法

1. 点击**添加文件**选择文件，或**添加文件夹**添加整个目录中的文件
2. 也可直接将文件拖放到界面上
3. 选择粉碎方式：快速、标准或深度
4. 检查文件列表，移除不想粉碎的文件
5. 点击**开始粉碎**并确认不可逆操作
6. 监控进度 — 粉碎成功的文件会从列表中移除，失败的文件保留以便查看
