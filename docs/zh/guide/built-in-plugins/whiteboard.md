# tinker-whiteboard

[TINKER](https://github.com/liriliri/tinker) 的白板插件，基于 [tldraw](https://tldraw.com/) 提供自由绘图和图表功能。

![Screenshot](/whiteboard.png)

## 功能特性

- **自由绘图** — 在无限画布上自由绘画、素描和标注
- **图形工具** — 创建矩形、椭圆、箭头、线条等形状
- **文本** — 在画布任意位置添加文本标签
- **便签** — 添加便签进行快速标注
- **画笔与橡皮擦** — 使用画笔绘制，使用橡皮擦擦除
- **选择与变换** — 选择、移动、缩放和旋转对象
- **撤销/重做** — 支持完整的撤销和重做操作
- **导出** — 将白板导出为图片

## 使用方法

1. 打开插件即可在空白无限画布上开始创作
2. 使用工具栏选择绘图工具：画笔、图形、文本、便签等
3. 在画布上绘制和排列元素
4. 使用撤销/重做修正操作
