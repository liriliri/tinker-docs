# tinker-music-player

[TINKER](https://github.com/liriliri/tinker) 的本地音乐播放器插件，支持播放列表管理和多种音频格式。

![Screenshot](/music_player.png)

## 功能特性

- **多格式支持** — 播放 MP3、FLAC、WAV、OGG、M4A、AAC 和 WMA 音频文件
- **播放列表管理** — 创建自定义播放列表，添加/移除曲目，管理收藏列表
- **播放模式** — 顺序播放、单曲循环和随机播放模式
- **播放队列** — 独立于播放列表的播放队列管理
- **歌词显示** — 自动加载 LRC 歌词并同步显示
- **音乐详情视图** — 在专用面板中查看专辑封面和歌词
- **最近播放** — 自动记录最近播放的曲目（最多 100 首）
- **文件搜索** — 直接在播放器中搜索并添加本地音频文件
- **迷你模式** — 紧凑的播放器窗口，占用最少屏幕空间
- **浮动歌词** — 在其他应用上方的浮动窗口中显示歌词
- **音量控制** — 可调节音量并持久保存设置

## 使用方法

### 添加音乐

1. 点击工具栏中的**添加**按钮，从电脑中选择音频文件
2. 或者使用搜索栏查找系统中的音频文件并直接添加
3. 曲目会自动导入元数据（标题、艺术家、专辑、封面）

### 播放音乐

1. 双击列表中的曲目开始播放
2. 使用底部播放栏控制播放（播放/暂停、上一首、下一首）
3. 点击进度条跳转到指定位置
4. 点击播放模式图标在顺序、循环和随机模式之间切换

### 管理播放列表

1. 使用侧边栏在**本地**、**最近**、**收藏**和自定义播放列表之间切换
2. 右键点击曲目将其添加到收藏或自定义播放列表
3. 点击侧边栏中的 **+** 按钮创建新播放列表
