# tinker-photo-develop

专业照片调色插件，基于 WebGL 实时渲染，支持曝光、色调、色彩、曲线、效果和细节等全面的调整功能。

![截图](/photo_develop.png)

## 功能特性

- **基本调整** — 控制曝光、亮度、对比度、高光、阴影、白色和黑色，实时预览
- **曲线** — 通过亮度、红、绿、蓝四个通道精细调整色调范围
- **颜色调整** — 调整色温、色调、自然饱和度和饱和度
- **混色器** — 精确控制 8 个颜色通道（红、橙、黄、绿、青、蓝、紫、洋红）的色相和明度
- **效果** — 添加暗角效果，可调整数量、中点、圆度和羽化；应用胶片颗粒，可控制大小和粗糙度
- **细节** — 锐化图片并调整阈值；减少亮度和颜色噪点
- **滤镜** — 应用 15+ 种预设滤镜，带实时缩略图预览，包括怀旧、鲜明、低保真、清新、胶片等
- **直方图** — 实时直方图可视化，辅助曝光分析
- **撤销/重做** — 完整的调整历史记录，支持防抖提交和撤销/重做
- **EXIF 保留** — 保存 JPEG 时保留原始 EXIF 元数据
- **拖放** — 支持直接拖放图片到编辑器中

## 使用方法

1. 点击**打开图片**选择照片，或直接将图片拖放至编辑器
2. 使用左侧面板的滑块调整图片，按分类组织：基本、曲线、颜色、混色器、效果、细节
3. 浏览**滤镜**标签页，通过实时预览应用预设滤镜
4. 使用**撤销**/**重做**浏览调整历史
5. 右上角的直方图显示实时曝光分布
6. 点击**保存图片**将编辑后的图片导出为 PNG、JPG 或 WebP 格式
7. 启用**覆盖原图**直接保存到原始文件
