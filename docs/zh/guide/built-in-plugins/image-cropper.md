# tinker-image-cropper

图片裁剪插件，支持旋转、翻转、缩放和宽高比控制。

![截图](/image_cropper.png)

## 功能特性

- **交互式裁剪** — 拖拽选择裁剪区域，实时预览
- **宽高比预设** — 自由、原始、1:1、4:3、3:2、16:9
- **自定义裁剪尺寸** — 以像素为单位设置精确裁剪尺寸
- **旋转** — 左旋或右旋 90 度
- **翻转** — 水平或垂直翻转图片
- **调整大小** — 将图片调整为指定尺寸，可选锁定宽高比
- **屏幕截图** — 直接截取屏幕并裁剪
- **剪贴板支持** — 从剪贴板粘贴图片，复制裁剪结果
- **撤销/重做** — 完整的裁剪操作历史支持
- **保存选项** — 覆盖原文件或另存为新文件

## 使用方法

1. 点击**打开**加载图片，或截取屏幕，或从剪贴板粘贴
2. 在图片上拖拽定义裁剪区域
3. 可选择**宽高比**预设或设置自定义裁剪尺寸
4. 根据需要使用旋转、翻转或调整大小控件
5. 点击**裁剪**应用裁剪
6. 点击**复制**将结果复制到剪贴板，或点击**保存**保存裁剪后的图片
