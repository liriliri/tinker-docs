# tinker-json-editor

JSON 编辑插件，提供文本和树形两种编辑模式，基于 Monaco Editor 和 JSONEditor。

![截图](/json_editor.png)

## 功能特性

- **文本模式** — 通过 Monaco Editor 编辑 JSON，支持语法高亮、验证和错误检测
- **树形模式** — 在交互式树形视图中浏览和编辑 JSON
- **格式化与压缩** — 格式化 JSON 缩进或压缩为紧凑形式
- **展开/折叠** — 在树形模式下展开或折叠所有节点
- **文件操作** — 新建文件、打开现有 JSON 文件、保存和另存为
- **未保存更改检测** — 跟踪修改状态，显示文件更改指示
- **撤销/重做** — 完整的撤销和重做支持
- **剪贴板支持** — 将 JSON 复制到剪贴板或从剪贴板粘贴
- **行数统计** — 显示编辑器中的行数

## 使用方法

1. 点击**打开**加载 JSON 文件，或直接在编辑器中输入
2. 使用工具栏切换**文本**和**树形**模式
3. 点击**格式化**美化 JSON，或点击**压缩**使其紧凑
4. 在树形模式下，使用**展开全部** / **折叠全部**浏览结构
5. 点击**保存**将更改保存回文件
