# tinker-system-info

[TINKER](https://github.com/liriliri/tinker) 的系统信息查看插件，显示计算机的详细硬件和软件信息。

![Screenshot](/system_info.png)

## 功能特性

- **系统概览** — 查看系统型号、制造商和序列号
- **操作系统信息** — 显示操作系统名称、版本和发布日期
- **处理器详情** — 显示 CPU 芯片名称、核心数、线程数和主频
- **内存信息** — 查看总内存、已使用和可用内存及内存类型
- **显卡与显示器** — 列出显卡和显示器分辨率
- **存储设备** — 显示磁盘布局，包括容量、类型和厂商信息
- **音频设备** — 列出可用的音频设备
- **网络接口** — 显示网络适配器的 MAC 地址、IPv4 和 IPv6 地址
- **点击复制** — 点击任意值即可复制到剪贴板

## 使用方法

1. 打开插件后自动加载并显示所有系统信息
2. 浏览各个分类：**概览**、**处理器**、**内存**、**显卡**、**显示器**、**存储**、**音频**和**网络**
3. 点击任意值即可复制到剪贴板
