# tinker-video-player

[TINKER](https://github.com/liriliri/tinker) 的视频播放插件，支持常见视频格式，提供播放控制、播放列表历史和媒体信息查看功能。

![Screenshot](/video_player.png)

## 功能特性

- **视频播放** — 支持播放 MP4、WebM、OGG、MOV、AVI、MKV 和 M4V 格式的视频文件
- **拖放打开** — 直接将视频文件拖放到播放器即可打开
- **播放控制** — 播放、暂停、重播、快进/快退和音量调节
- **播放速度** — 切换播放倍速，支持快放和慢放
- **画中画** — 在浮动迷你窗口中观看视频
- **全屏模式** — 进入全屏获得沉浸式观看体验
- **播放列表历史** — 自动记录播放历史，显示缩略图和时长
- **续播功能** — 记住播放进度，从上次中断处继续播放
- **媒体信息** — 查看详细的文件信息，包括编解码器、分辨率、FPS、比特率和音频采样率
- **字幕支持** — 可用时启用或禁用字幕

## 使用方法

1. 点击**打开视频**或将视频文件拖放到播放器中
2. 使用播放控件进行播放、暂停、快进快退和音量调节
3. 点击**倍速**按钮更改播放速度
4. 点击**画中画**按钮悬浮播放视频，或点击**全屏**按钮进入全屏模式
5. 点击**文件信息**查看详细的媒体信息
6. 打开**播放列表**面板浏览播放历史，快速切换视频
