# tinker-unit-converter

[TINKER](https://github.com/liriliri/tinker) 的单位换算插件，支持七大测量类别的公制和英制单位换算。

![Screenshot](/unit_converter.png)

## 功能特性

- **长度** — 千米、米、分米、厘米、毫米、微米、英寸、英尺、码、英里之间互转
- **面积** — 平方千米、平方米、平方厘米、英亩、平方英尺之间互转
- **体积** — 升、毫升、加仑之间互转
- **重量** — 千克、克、毫克、磅、盎司之间互转
- **温度** — 摄氏度、华氏度、开尔文之间互转
- **时间** — 天、小时、分钟、秒、毫秒之间互转
- **数据** — 字节、千字节、兆字节、吉字节、太字节之间互转
- **实时换算** — 输入数值时所有单位结果即时更新
- **高精度计算** — 基于 mathjs BigNumber 提供精确计算
- **复制到剪贴板** — 点击任意结果值即可复制

## 使用方法

1. 从工具栏选择测量类别：**长度**、**面积**、**体积**、**重量**、**温度**、**时间**或**数据**
2. 选择源单位并在输入框中输入数值
3. 该类别下的所有其他单位会实时显示转换结果
4. 点击任意结果即可复制到剪贴板
