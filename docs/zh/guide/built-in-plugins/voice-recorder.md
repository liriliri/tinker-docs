# tinker-voice-recorder

[TINKER](https://github.com/liriliri/tinker) 的录音机插件，支持通过麦克风录制音频，提供波形可视化和录音回放预览。

![Screenshot](/voice_recorder.png)

## 功能特性

- **音频录制** — 一键通过麦克风录制音频
- **波形可视化** — 录制过程中实时显示声音波形
- **暂停/继续** — 暂停和继续录制，不丢失已录内容
- **录制计时** — 实时显示当前录制时长
- **预览回放** — 保存前可回放录制的音频，显示波形
- **保存录音** — 将录音保存为文件，支持自定义文件名
- **下载** — 下载录制的音频文件
- **删除录音** — 丢弃当前录音并重新开始

## 使用方法

1. 点击**录制**按钮开始通过麦克风录音
2. 在浏览器提示时授予麦克风访问权限
3. 根据需要使用**暂停**和**继续**控制录制
4. 点击**停止**完成录制
5. 使用内置播放控件预览录音
6. 点击**保存**将录音保存为文件，或点击**删除**丢弃录音
