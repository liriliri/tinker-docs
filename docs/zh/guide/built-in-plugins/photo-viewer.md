# tinker-photo-viewer

照片查看插件，支持浏览本地目录中的照片，按日期组织并以瀑布流布局展示，同时可查看完整的 EXIF 元数据。

![截图](/photo_viewer.png)

## 功能特性

- **瀑布流画廊** — 以响应式瀑布流网格浏览照片，按日期分组展示
- **照片查看器** — 全尺寸查看照片，支持上一张/下一张导航和平滑过渡
- **EXIF 元数据** — 查看详细的 EXIF 信息，包括相机信息、拍摄参数、GPS 位置和技术参数
- **文件夹扫描** — 扫描选定的本地目录，自动导入照片并保持同步
- **文件导入** — 通过文件对话框手动导入照片，支持多选
- **搜索** — 按文件名快速搜索系统内的照片
- **在文件夹中显示** — 在系统文件管理器中打开照片所在位置

## 使用方法

1. 点击**导入**按钮手动添加照片，或点击**扫描文件夹**导入整个目录
2. 照片会按日期分组以瀑布流布局展示
3. 点击任意照片可在全屏查看器中打开
4. 使用方向键或导航按钮浏览照片
5. 右侧面板显示当前照片的 EXIF 元数据，包括相机、镜头、曝光设置和 GPS 位置
6. 点击**在文件夹中显示**可在磁盘上定位原始文件
