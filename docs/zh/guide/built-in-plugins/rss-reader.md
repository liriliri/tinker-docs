# tinker-rss-reader

[TINKER](https://github.com/liriliri/tinker) 的 RSS/Atom 订阅阅读器插件，支持订阅源管理、文章浏览和内容阅读。

![Screenshot](/rss_reader.png)

## 功能特性

- **订阅源管理** — 通过 URL 添加 RSS/Atom 订阅源，管理多个来源
- **文章列表** — 按日期排序浏览文章，显示标题、摘要和缩略图预览
- **已读/未读追踪** — 打开文章时自动标记为已读，每个订阅源显示未读数量
- **文章详情视图** — 直接在插件内阅读格式化的文章内容
- **加载全文** — 当订阅源仅提供摘要时，可从原始页面获取完整文章内容
- **搜索文章** — 通过标题和摘要搜索文章，快速查找内容
- **状态过滤** — 筛选显示全部文章或仅显示未读文章
- **列表和卡片视图** — 在列表视图和卡片视图之间切换浏览文章
- **全部标记为已读** — 将当前订阅源或所有订阅源的文章标记为已读
- **在浏览器中打开** — 在默认浏览器中打开文章查看完整网页
- **可折叠侧边栏** — 切换订阅源侧边栏以最大化阅读空间
- **刷新订阅** — 刷新单个或全部订阅源以获取新文章

## 使用方法

1. 点击 **+** 按钮添加新的 RSS/Atom 订阅源 URL
2. 在侧边栏选择一个订阅源查看其文章，或选择**全部订阅**查看所有文章
3. 点击文章在详情面板中阅读内容
4. 使用**未读**过滤器仅查看新文章
5. 如果订阅源仅显示摘要，点击**加载全文**获取完整内容
6. 使用搜索栏按标题或内容查找文章
7. 点击**全部标记为已读**清除未读计数
8. 使用工具栏按钮在**列表视图**和**卡片视图**之间切换
