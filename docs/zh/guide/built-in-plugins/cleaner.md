# tinker-cleaner

[TINKER](https://github.com/liriliri/tinker) 垃圾清理插件，扫描和清理系统垃圾文件以释放磁盘空间。

![Screenshot](/cleaner.png)

## 功能

- **分类扫描** — 将垃圾文件按类别分组：系统、用户缓存、浏览器、开发工具和应用程序
- **全面的规则覆盖** — 检测系统临时文件、日志、崩溃报告、浏览器缓存、npm/Yarn/pip/Cargo 缓存、VS Code 缓存等
- **大小分析** — 显示每个检测到的垃圾项的大小和每个类别的总大小
- **选择性清理** — 选择单个项目或全选进行批量清理
- **移到回收站选项** — 将文件安全移动到回收站而非永久删除
- **扫描进度** — 扫描时实时显示进度和当前路径信息
- **确认对话框** — 清理前确认，显示已选项目数量和总大小摘要

## 使用方法

1. 插件打开时自动开始扫描
2. 扫描完成后，查看按类别组织的检测到的垃圾文件
3. 使用侧边栏按类别筛选（全部、系统、用户缓存、浏览器、开发工具、应用程序）
4. 选择或取消选择要清理的项目，或使用**全选**
5. 如果希望将文件移到回收站而非永久删除，启用**移到回收站**选项
6. 点击**清理**开始清理选中的项目
7. 在确认对话框中确认操作
8. 随时点击**扫描**重新扫描系统
