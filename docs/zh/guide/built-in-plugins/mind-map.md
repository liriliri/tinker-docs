# tinker-mind-map

[TINKER](https://github.com/liriliri/tinker) 的思维导图插件，支持创建、编辑和组织思维导图，提供多种布局方式。

![Screenshot](/mind_map.png)

## 功能特性

- **多种布局** — 支持逻辑结构图、思维导图、组织结构图和目录组织图之间切换
- **大纲侧边栏** — 以树形结构查看和导航节点
- **节点操作** — 通过右键菜单插入子节点/同级节点、移动、复制、剪切、粘贴和删除节点
- **文件操作** — 新建思维导图、打开已有文件（JSON/SMM）以及保存更改
- **撤销/重做** — 支持对所有编辑操作的撤销和重做
- **缩放控制** — 支持放大/缩小以及预设缩放值，或将整个导图适应画布
- **导出图片** — 将思维导图导出为 PNG 图片
- **键盘导航** — 使用键盘快捷键导航和编辑节点
- **超链接和备注** — 为节点添加超链接和备注

## 使用方法

1. 启动插件后会显示一个包含根节点和示例分支的默认思维导图
2. 双击节点即可编辑文本内容
3. 右键点击节点可打开上下文菜单，进行插入、移动、复制或删除等操作
4. 使用工具栏中的**布局选择器**切换不同的布局样式
5. 切换**侧边栏**可查看大纲并快速导航到任意节点
6. 点击**保存**将思维导图保存到文件，或点击**打开**加载已有文件
7. 使用右下角的缩放控件调整视图，或点击**适应画布**将导图适配到画布
8. 点击**导出图片**将思维导图下载为 PNG 文件
