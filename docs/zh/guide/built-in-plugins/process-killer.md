# tinker-process-killer

[TINKER](https://github.com/liriliri/tinker) 的进程管理插件，支持查看、搜索和终止系统运行中的进程。

![Screenshot](/process_killer.png)

## 功能特性

- **进程列表** — 查看所有运行中的进程，包括名称、PID、CPU 占用、内存占用、用户和状态
- **视图模式** — 在 CPU、内存和端口视图之间切换，关注不同的资源指标
- **搜索** — 按进程名称、PID、用户或端口号筛选进程
- **排序** — 按任意列进行升序或降序排序
- **终止进程** — 通过确认对话框终止指定进程
- **网络端口** — 显示每个进程的监听端口和协议
- **进程图标** — 为每个进程显示应用程序图标
- **自动刷新** — 按固定间隔自动刷新进程列表
- **手动刷新** — 按需手动刷新进程列表

## 使用方法

1. 启动插件后会显示按 CPU 占用排序的所有运行中进程列表
2. 使用工具栏切换 **CPU**、**内存**和**端口**视图模式
3. 使用**搜索栏**按名称、PID 或端口筛选进程
4. 点击列标题按对应字段排序
5. 点击进程行上的**终止**按钮并确认以终止该进程
6. 切换**自动刷新**以保持列表自动更新
