# tinker-audio-editor

一个功能丰富的音频编辑插件，适用于 [TINKER](https://github.com/liriliri/tinker)，支持多格式音频编辑与波形可视化。

![截图](/audio_editor.png)

## 功能特性

- **多格式支持** — 打开 MP3、WAV、OGG、FLAC、M4A、AAC、OPUS 和 WebM 音频文件
- **波形可视化** — 立体声波形显示，支持缩放、滚动和时间轴
- **区域选择** — 拖动选择音频区域进行编辑操作
- **基础编辑** — 裁剪、删除、静音、复制、剪切和粘贴音频
- **信号处理** — 增益调整、标准化、变速、淡入/淡出
- **插入静音** — 在任意位置添加静音段（1–60 秒）
- **实时监控** — VU 音量表带峰值指示器，支持单声道静音
- **撤销/重做** — 完整的历史记录支持，最多 20 级撤销
- **键盘快捷键** — Space（播放/暂停）、Ctrl+Z（撤销）、Ctrl+A（全选）等
- **导出 WAV** — 将编辑后的音频保存为 WAV 文件

## 使用方法

### 打开音频

1. 点击工具栏中的**打开**按钮，或将音频文件拖放到窗口中
2. 音频波形将与时间轴一起显示

### 编辑音频

1. 在波形上拖动以选择一个区域
2. 使用工具栏按钮或右键菜单执行**裁剪**、**删除**、**静音**、**增益**、**标准化**、**变速**、**淡入**或**淡出**等操作
3. 使用**复制**、**剪切**和**粘贴**在不同位置之间移动音频
4. 使用 **Ctrl+Z** 撤销，**Ctrl+Y** 重做

### 播放

1. 按 **Space** 或点击**播放**按钮开始播放
2. 使用定位按钮向前/向后导航
3. 在波形底部的 VU 音量表中监控音频电平
4. 点击 **L** 或 **R** 标签来静音单个声道

### 导出

1. 点击工具栏中的**导出**按钮
2. 选择保存位置和文件名
3. 音频将保存为 WAV 文件
