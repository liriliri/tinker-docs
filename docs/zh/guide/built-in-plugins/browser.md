# tinker-browser

网页浏览器插件，支持多标签页和搜索。

![截图](/browser.png)

## 功能特性

- **多标签页浏览** — 在多个标签页中打开不同网页，支持拖拽排序标签页
- **地址栏搜索** — 输入 URL 直接导航，或输入关键词通过 Google 搜索
- **导航控件** — 后退、前进、刷新和停止按钮，支持标准浏览操作
- **新标签页快捷方式** — 在新标签页中添加常用网站快捷方式，方便快速访问
- **站点管理** — 添加、编辑和删除网站快捷方式，自动获取网站图标
- **标签页管理** — 新建标签页、关闭标签页和切换标签页
- **网站图标显示** — 在标签页和快捷方式中显示网站图标
- **加载指示器** — 页面加载时显示视觉反馈

## 使用方法

1. 点击标签栏上的 **+** 按钮打开新标签页
2. 在地址栏中输入 URL 或搜索关键词，按 **Enter** 键
3. 使用工具栏中的后退、前进和刷新按钮进行导航
4. 在新标签页中，点击 **+** 按钮添加网站快捷方式
5. 右键点击网站快捷方式可编辑或删除
6. 点击网站快捷方式导航到该网站
7. 拖拽标签页可重新排列顺序
