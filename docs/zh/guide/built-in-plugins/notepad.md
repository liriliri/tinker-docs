# tinker-notepad

[TINKER](https://github.com/liriliri/tinker) 的轻量级文本编辑插件，基于 Monaco Editor 提供语法高亮和语言检测功能。

![Screenshot](/notepad.png)

## 功能特性

- **Monaco Editor** — 支持语法高亮、行号显示和自动换行的文本编辑
- **语言检测** — 根据文件扩展名自动检测并应用对应的语言高亮
- **文件操作** — 新建文件、打开已有文本文件、保存及另存为
- **未保存更改检测** — 跟踪文件修改状态
- **撤销/重做** — 完整的撤销和重做支持
- **搜索** — 基于 Monaco Editor 的内置搜索功能
- **剪贴板支持** — 复制内容或从剪贴板粘贴
- **字体大小调节** — 支持放大和缩小编辑器字体
- **光标位置** — 在工具栏中显示当前行号和列号

## 使用方法

1. 启动插件即可在空白文档中开始编辑
2. 点击**打开**加载文本文件，或点击**新建**创建空白文档
3. 使用 Monaco Editor 提供的语法高亮进行文本编辑
4. 使用**搜索**按钮或键盘快捷键在文档中查找文本
5. 使用**放大**和**缩小**按钮调节字体大小
6. 点击**保存**将更改保存到文件
