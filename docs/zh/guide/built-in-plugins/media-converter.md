# tinker-media-converter

媒体格式转换插件，基于 FFmpeg，支持批量转换视频、音频和图片文件。

![截图](/media_converter.png)

## 功能特性

- **视频转换** — 在 MP4、MKV、WebM、MOV、AVI、FLV、TS、WMV、3GP 和 GIF 之间转换，支持多种编码器（H.264、H.265、VP9、AV1、ProRes、Xvid 等）
- **音频转换** — 在 MP3、M4A、AAC、OGG、FLAC、WAV 和 Opus 之间转换
- **图片转换** — 在 JPG、PNG、WebP、BMP、TIFF 和 AVIF 之间转换
- **批量处理** — 一次转换多个文件，显示进度跟踪
- **格式自动检测** — 跳过已为目标格式的文件
- **自定义输出目录** — 将转换文件保存到指定目录或与原文件同目录
- **媒体信息显示** — 查看每个文件的分辨率、帧率、时长、比特率和采样率
- **取消支持** — 随时取消转换任务

## 使用方法

1. 在工具栏中切换**视频**、**音频**和**图片**模式
2. 点击**打开**或拖拽媒体文件添加
3. 从下拉菜单中选择目标输出**格式**
4. 可选设置输出目录
5. 点击**转换**开始批量转换
6. 监控每个文件的进度；点击**取消**可随时停止
