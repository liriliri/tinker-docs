# tinker-qrcode

[TINKER](https://github.com/liriliri/tinker) 的二维码生成与扫描插件，支持自定义二维码生成和基于图片的二维码识别。

![Screenshot](/qrcode.png)

## 功能特性

- **二维码生成** — 根据任意文本或 URL 实时生成二维码
- **自定义尺寸** — 从预设尺寸（300–600）中选择或输入自定义尺寸
- **颜色自定义** — 设置二维码的前景色和背景色
- **纠错等级** — 支持 L、M、Q、H 四种纠错等级
- **自定义图标** — 在二维码中心添加图标/Logo
- **二维码扫描** — 从图片文件中识别二维码
- **复制到剪贴板** — 将生成的二维码图片复制到剪贴板
- **保存为图片** — 将二维码保存为图片文件

## 使用方法

1. 在左侧输入区域输入文本或 URL
2. 右侧会实时自动生成对应的二维码
3. 根据需要调整**尺寸**、**前景色**、**背景色**和**纠错等级**
4. 点击**设置图标**可在二维码中心添加自定义 Logo
5. 点击**复制**将二维码复制到剪贴板，或点击**保存**导出为图片文件
6. 点击**扫描二维码**并选择图片即可从中识别二维码内容
