# tinker-calculator

科学计算器插件，提供基础算术运算和高级科学计算功能。

![截图](/calculator.png)

## 功能特性

- **基础算术运算** — 加、减、乘、除
- **科学函数** — 平方、立方、平方根、立方根、倒数、阶乘、随机数生成
- **角度/弧度模式** — 在角度和弧度模式之间切换，用于三角函数计算
- **第二功能模式** — 通过切换第二功能键访问更多函数
- **存储功能** — M+、M-、MC 和 MR，用于存储和调用数值
- **表达式预览** — 在结果显示上方查看当前表达式
- **状态持久化** — 表达式、存储值和模式设置跨会话保留

## 使用方法

1. 点击数字和运算符按钮来构建表达式
2. 按 `=` 求值
3. 使用科学函数按钮（x²、√、x! 等）进行高级计算
4. 切换 `Deg/Rad` 在角度和弧度模式之间切换
5. 使用存储按钮（M+、M-、MR、MC）存储和调用数值
