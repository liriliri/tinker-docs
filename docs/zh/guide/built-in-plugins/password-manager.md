# tinker-password-manager

[TINKER](https://github.com/liriliri/tinker) 的密码管理插件，基于 KeePass（KDBX）格式实现安全的密码存储与管理。

![Screenshot](/password_manager.png)

## 功能特性

- **KeePass 兼容** — 创建、打开和保存与 KeePass 兼容的 KDBX 数据库文件
- **分组管理** — 通过层级分组来组织条目，支持创建、重命名和删除分组
- **条目管理** — 为每个条目存储标题、用户名、密码、URL、备注和自定义字段
- **搜索** — 按标题、用户名、URL 或备注快速搜索所有条目
- **密码可见性** — 支持显示/隐藏密码切换
- **复制到剪贴板** — 一键复制用户名或密码
- **密钥文件支持** — 支持使用密码和可选密钥文件打开数据库
- **最近文件** — 快速访问最近打开的数据库文件
- **数据库锁定** — 锁定数据库以保护数据安全

## 使用方法

1. 点击**新建数据库**创建新的密码数据库，或点击**打开数据库**加载已有的 KDBX 文件
2. 输入**主密码**（及可选的密钥文件）解锁数据库
3. 使用左侧的**分组树**将条目分类整理
4. 点击**创建条目**添加新的密码条目，填写标题、用户名、密码、URL 和备注
5. 选择条目即可在右侧面板中查看或编辑详细信息
6. 使用**搜索**栏在所有分组中快速查找条目
7. 点击**保存**将更改持久化到数据库文件
