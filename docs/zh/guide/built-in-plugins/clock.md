# tinker-clock

时钟插件，提供三种显示风格：翻页时钟、模拟时钟和数字时钟，支持时区切换。

![截图](/clock.png)

## 功能特性

- **翻页时钟** — 经典翻页动画风格显示
- **模拟时钟** — 玻璃拟态设计，指针平滑动画
- **数字时钟** — 七段 LED 风格显示，SVG 动画效果
- **时区支持** — 在本地时间和不同时区之间切换
- **全屏模式** — 全屏显示时钟
- **双语界面** — 支持中文和英文

## 使用方法

1. 从工具栏选择时钟风格（翻页、模拟或数字）
2. 使用时区选择器切换显示的时区
3. 点击全屏按钮进入全屏模式
