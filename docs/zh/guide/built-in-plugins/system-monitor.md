# tinker-system-monitor

[TINKER](https://github.com/liriliri/tinker) 的实时系统监控插件，跟踪计算机的 CPU、内存、磁盘和网络使用情况。

![Screenshot](/system_monitor.png)

## 功能特性

- **CPU 使用率** — 通过百分比环形图和历史曲线实时查看 CPU 负载
- **内存使用** — 跟踪活动内存与总内存的占用情况
- **网络吞吐** — 监控下载（RX）和上传（TX）速率
- **磁盘 I/O** — 显示磁盘读写速度
- **磁盘空间** — 列出每个挂载卷的容量、已用空间和使用百分比
- **电池状态** — 在配备电池的设备上显示电量
- **系统运行时间** — 显示系统已经运行的时长
- **暂停/恢复** — 随时暂停采样并恢复
- **悬浮窗口** — 将监控弹出为小型置顶窗口，方便随时查看

## 使用方法

1. 打开插件即可开始采集指标，采样几次后图表会填满
2. 在 **CPU**、**内存**、**网络** 和 **磁盘 I/O** 面板中查看实时变化趋势
3. 向下滚动查看每个挂载卷的 **磁盘空间** 信息
4. 点击工具栏中的 **暂停** 按钮可冻结采样，再次点击恢复
5. 点击 **悬浮** 按钮可打开紧凑的悬浮窗口，包含 CPU 和内存的环形图以及实时的网络/磁盘数值
