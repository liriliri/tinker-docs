# tinker-disk-usage

磁盘空间分析插件，通过交互式树状图可视化存储空间使用情况，帮助识别占用空间较大的文件和目录。

![截图](/disk_usage.png)

## 功能特性

- **文件夹扫描** — 选择任意文件夹分析其磁盘使用情况，支持配置扫描深度
- **交互式树状图** — 基于 D3 的可视化，方块大小与磁盘占用成正比
- **文件类型着色** — 为 JS/TS、JSON、CSS、HTML、图片、markdown、目录等使用不同颜色
- **逐层浏览** — 单击展开目录，双击进入目录
- **前进/后退/上级导航** — 类浏览器的历史记录，方便探索
- **懒加载** — 深层嵌套目录按需加载，提升性能
- **右键菜单操作** — 右键可在文件管理器中显示或删除文件/文件夹
- **扫描进度** — 扫描过程中实时显示文件数量、总大小和错误数
- **重新扫描** — 刷新当前目录的磁盘使用数据
- **深色/浅色主题支持** — 可视化颜色自动适配当前主题

## 使用方法

1. 点击"打开文件夹"按钮选择要分析的目录
2. 等待扫描完成，进度会实时显示
3. 浏览树状图：较大的方块代表占用空间更多的文件或目录
4. 单击目录展开，双击进入查看详情
5. 使用工具栏的前进、后退和上级按钮进行导航
6. 右键点击任意项目可在文件管理器中显示或删除
