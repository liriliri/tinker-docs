# tinker-float

[TINKER](https://github.com/liriliri/tinker) 悬浮窗插件，可在置顶悬浮窗中显示网页、图片、视频和文本。

![Screenshot](/float.png)

## 功能

- **网页显示** — 输入 URL 即可在悬浮窗中加载并显示任意网页
- **图片预览** — 打开图片文件（PNG、JPG、GIF、WebP、BMP、SVG）或粘贴图片 URL 进行悬浮显示
- **视频播放** — 在悬浮窗中播放视频文件（MP4、WebM、MOV、AVI、MKV 等）
- **文本显示** — 打开文本文件（TXT、MD、JSON、JS、TS、HTML、CSS 等）进行悬浮参考
- **屏幕截图** — 截取当前屏幕并在悬浮窗中显示
- **始终置顶** — 使悬浮窗保持在所有其他窗口之上，方便参考
- **自定义窗口大小** — 调整悬浮窗的宽度和高度
- **自动宽高比** — 图片悬浮窗自动保持正确的宽高比

## 使用方法

1. 点击**相机**图标截取屏幕，或点击**文件夹**图标打开文件
2. 也可以在地址栏输入 URL 并按 **Enter** 加载网页
3. 点击尺寸显示（例如 "400 x 300"）调整窗口大小
4. 切换**置顶**复选框控制悬浮窗是否始终在最上层
5. 点击**浮窗**按钮启动悬浮窗
6. 按 **Esc** 关闭悬浮窗
