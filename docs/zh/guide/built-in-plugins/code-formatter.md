# tinker-code-formatter

代码格式化插件，支持多种编程语言和可配置的缩进方式。

![截图](/code_formatter.png)

## 功能特性

- **多语言支持** — 支持 JavaScript、TypeScript、HTML、CSS 和 JSON
- **可配置缩进** — 可选无缩进、2、4、6 或 8 空格
- **代码编辑器** — 基于 Monaco 的编辑器，支持语法高亮和行号显示
- **快捷操作** — 一键格式化、复制、粘贴和清除
- **双语界面** — 支持中文和英文

## 使用方法

1. 从下拉菜单中选择代码语言
2. 粘贴或输入代码到编辑器中
3. 选择偏好的缩进宽度
4. 点击格式化按钮格式化代码
5. 使用复制按钮将格式化结果复制到剪贴板
