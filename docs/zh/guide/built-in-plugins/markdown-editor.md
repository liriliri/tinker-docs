# tinker-markdown-editor

Markdown 编辑插件，基于 Monaco Editor，支持实时预览和分栏视图。

![截图](/markdown_editor.png)

## 功能特性

- **Monaco Editor** — 通过 Monaco Editor 编辑 Markdown，支持语法高亮
- **实时预览** — 实时渲染 Markdown 预览，支持同步滚动
- **视图模式** — 分栏视图、仅编辑器、仅预览三种模式
- **文件操作** — 新建文件、打开现有 Markdown 文件、保存和另存为
- **未保存更改检测** — 跟踪修改状态，显示文件更改指示
- **撤销/重做** — 完整的撤销和重做支持
- **剪贴板支持** — 复制 Markdown 内容或从剪贴板粘贴
- **行数统计** — 显示编辑器中的行数

## 使用方法

1. 点击**打开**加载 Markdown 文件，或点击**新建**创建空白文档
2. 在左侧编辑器面板中编写 Markdown
3. 右侧的渲染预览会实时更新
4. 使用工具栏切换**分栏视图**、**仅编辑器**和**仅预览**模式
5. 点击**保存**将更改保存回文件
