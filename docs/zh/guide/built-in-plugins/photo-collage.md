# tinker-photo-collage

[TINKER](https://github.com/liriliri/tinker) 的照片拼接插件，支持将多张照片排列到各种网格布局中。

![Screenshot](/photo_collage.png)

## 功能特性

- **网格布局模板** — 提供多种布局模板，支持 2 到 16 张照片
- **拖放操作** — 从侧边栏拖动照片到布局槽位中
- **自定义画布** — 配置画布宽度、高度和背景颜色
- **内边距、间距和圆角** — 调整内边距、槽位间距和圆角半径
- **照片调整** — 在每个槽位中缩放和重新定位照片
- **网格调整** — 拖动网格分隔线来调整行列大小
- **背景图片** — 为画布设置自定义背景图片
- **随机排列** — 随机打乱照片在槽位中的位置
- **保存和复制** — 将拼接结果保存为图片或复制到剪贴板

## 使用方法

1. 点击侧边栏的**照片**区域添加照片
2. 选择照片数量并选择**布局模板**
3. 将照片从侧边栏拖放到画布上的布局槽位中
4. 使用侧边栏控件调整**内边距**、**间距**和**圆角**
5. 拖动网格分隔线可调整行列大小
6. 在槽位内滚动可缩放照片，拖动可重新定位照片
7. 点击**随机**打乱照片位置
8. 点击**保存**将拼接结果导出为图片，或点击**复制图片**复制到剪贴板
