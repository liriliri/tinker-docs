# tinker-clipboard

剪贴板管理插件，实时监控并记录剪贴板历史，支持文本、图片和文件。

![截图](/clipboard.png)

## 功能特性

- **剪贴板监控** — 实时自动记录剪贴板变化
- **多种内容类型** — 支持文本、图片和文件条目
- **筛选和搜索** — 按内容类型（全部/文本/图片/文件）筛选，并搜索剪贴板历史
- **快速复制** — 点击任意条目即可重新复制到剪贴板
- **无限滚动** — 通过懒加载高效浏览剪贴板历史
- **重复检测** — 自动检测并避免重复条目
- **双语界面** — 支持中文和英文

## 使用方法

1. 插件会自动开始监控剪贴板
2. 复制任意文本、图片或文件，即可在历史记录中看到
3. 使用筛选标签仅显示文本、图片或文件条目
4. 使用搜索栏查找特定条目
5. 点击条目即可重新复制到剪贴板
6. 点击删除按钮移除单个条目，或使用清除按钮移除所有条目
