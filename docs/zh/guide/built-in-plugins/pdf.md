# tinker-pdf

[TINKER](https://github.com/liriliri/tinker) 的 PDF 查看插件，基于 PDF.js 实现 PDF 文档的渲染与浏览。

![Screenshot](/pdf.png)

## 功能特性

- **PDF 渲染** — 基于 PDF.js 的高质量 PDF 页面渲染
- **页面导航** — 通过上一页/下一页按钮或页码输入在页面间导航
- **缩放控制** — 支持放大、缩小和重置缩放级别
- **缩略图侧边栏** — 查看页面缩略图以快速导航
- **文档大纲** — 通过 PDF 的目录结构进行导航（如果文档包含大纲）
- **导出为图片** — 将所有页面导出为 PNG 图片，支持配置缩放比例
- **拖放打开** — 将 PDF 文件拖放到查看器即可打开
- **自动适配** — 自动将页面宽度适配到容器

## 使用方法

1. 点击**打开文件**或将 PDF 文件拖放到查看器中
2. 使用**上一页/下一页**按钮或滚动来浏览页面
3. 通过**放大**、**缩小**或**重置缩放**调整缩放级别
4. 切换**侧边栏**查看页面缩略图或文档大纲
5. 点击缩略图或大纲条目可跳转到对应页面
6. 点击**导出为图片**将所有页面导出为 PNG 文件到指定文件夹
