# tinker-ai-assistant

AI 助手插件，通过对话完成任务，可使用文件操作、Shell 命令和网页搜索等工具。

![截图](/ai_assistant.png)

## 功能特性

- **多提供商支持** — 支持多种 AI 提供商和模型
- **工具调用** — 执行 Shell 命令、读写编辑文件、列出目录、网页搜索和抓取
- **会话持久化** — 对话内容跨会话保存和恢复
- **工作目录** — 可设置自定义工作目录用于文件和 Shell 操作
- **双语界面** — 支持中文和英文

## 使用方法

1. 从下拉菜单中选择 AI 提供商和模型
2. 点击工具栏中的文件夹图标可设置工作目录（可选）
3. 在输入框中输入消息，按回车或点击发送按钮
4. 助手会回复并可能使用工具来帮你完成任务

## 可用工具

| 工具 | 描述 |
|------|------|
| **exec** | 执行 Shell 命令 |
| **read_file** | 读取文件内容 |
| **write_file** | 写入文件内容 |
| **edit_file** | 查找替换编辑文件 |
| **list_dir** | 列出目录内容 |
| **web_search** | 网页搜索 |
| **web_fetch** | 抓取网页内容 |
