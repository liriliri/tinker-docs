# tinker-terminal

[TINKER](https://github.com/liriliri/tinker) 的终端模拟器插件，支持 SSH 配置管理和分屏功能。

![Screenshot](/terminal.png)

## 功能特性

- **终端模拟器** — 基于 xterm.js 的全功能终端
- **多标签页** — 新建、关闭、拖拽排序和切换终端标签页
- **分屏** — 水平或垂直分割当前面板，支持预设布局（双列、三列、网格）
- **SSH 会话** — 通过密码或私钥认证连接远程服务器
- **会话管理** — 将 SSH 和本地会话组织到文件夹中，便于快速访问
- **自定义 Shell** — 为本地会话配置使用的 Shell
- **工作目录** — 为每个会话设置自定义起始目录
- **侧边栏** — 切换会话侧边栏，管理已保存的连接
- **面板标题跟踪** — 标签页标题根据活动面板的进程自动更新

## 使用方法

1. 打开插件即可启动一个新的本地终端会话
2. 使用标签栏创建更多终端标签页
3. 右键点击面板头部可水平或垂直分割
4. 打开侧边栏管理已保存的会话（本地或 SSH）
5. 创建文件夹，然后添加配置了自定义 Shell、工作目录或 SSH 连接信息的会话
6. 点击已保存的会话即可在新标签页中打开
