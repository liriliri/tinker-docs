# tinker-hosts

Hosts 文件管理插件，可创建、管理和快速切换多个 hosts 配置。

![截图](/hosts.png)

## 功能特性

- **查看系统 hosts** — 读取并显示当前系统 hosts 文件内容
- **多配置管理** — 创建和管理多个命名的 hosts 配置
- **快速切换** — 单击即可激活或停用配置
- **自动应用** — 活跃配置的更改会自动应用到系统 hosts 文件
- **配置管理** — 通过右键菜单重命名、编辑和删除配置
- **安全合并** — 托管内容使用标记包裹，不会影响现有的 hosts 条目

## 使用方法

1. 侧边栏中的**系统 Hosts** 视图显示当前系统 hosts 文件的内容
2. 点击**新建配置**创建命名的 hosts 配置（如"开发环境"、"测试环境"）
3. 在侧边栏中选择一个配置，在编辑器中编辑其 hosts 条目
4. 点击配置旁的指示点即可激活或停用该配置
5. 活跃的配置会自动合并并应用到系统 hosts 文件
6. 右键点击配置可进行重命名、删除或切换激活状态等操作

> **注意：** 写入系统 hosts 文件可能需要管理员权限。
