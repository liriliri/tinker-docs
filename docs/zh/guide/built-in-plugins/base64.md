# tinker-base64

Base64 编解码插件，支持文本和文件编码。

![截图](/base64.png)

## 功能特性

- **文本模式** — 编码和解码 Base64 字符串
- **文件模式** — 将文件编码为 Base64，可选 Data URL 输出
- **解码为文件** — 将 Base64 数据解码并保存为文件
- **剪贴板支持** — 从剪贴板粘贴文本或文件，点击输出即可复制
- **Web Worker** — 大文件编码在后台线程中运行，提升性能

## 使用方法

### 文本模式

1. 在工具栏中选择**文本**模式
2. 在输入面板中输入文本或 Base64 字符串
3. 点击**编码**将文本转换为 Base64，或点击**解码**将 Base64 转换回文本
4. 点击输出内容即可复制到剪贴板

### 文件模式

1. 在工具栏中选择**文件**模式
2. 选择要编码的文件
3. Base64 结果将显示在输出面板中
4. 可选开启 **Data URL** 以包含 MIME 类型前缀
5. 点击输出内容即可复制到剪贴板
