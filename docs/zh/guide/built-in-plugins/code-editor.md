# tinker-code-editor

[TINKER](https://github.com/liriliri/tinker) 的轻量级代码编辑器插件，支持语法高亮、多标签页和内置终端。

![Screenshot](/code_editor.png)

## 功能特性

- **语法高亮** — 基于 CodeMirror 的语言感知代码编辑
- **多标签页编辑** — 在标签页中打开多个文件，支持拖拽排序
- **文件树侧边栏** — 浏览项目目录并点击打开文件
- **未保存状态追踪** — 标签页上标记未保存的更改
- **内置终端** — 切换内置终端面板，支持多标签页和分屏
- **终端分屏** — 水平或垂直分割终端，支持预设布局（双列、三列、网格）
- **保存文件** — 将编辑后的文件保存到磁盘
- **光标位置** — 状态栏显示当前行号和列号
- **状态持久化** — 记住上次打开的文件夹、侧边栏和终端可见性

## 使用方法

1. 点击 **打开文件夹** 选择项目目录
2. 在侧边栏文件树中浏览并点击文件，在新标签页中打开
3. 在编辑器中编辑代码——未保存的更改会在标签页上标记
4. 按 **Ctrl+S**（或使用保存操作）将更改写入磁盘
5. 从工具栏切换内置终端，在项目中运行命令
6. 使用终端的分屏选项并行执行命令
