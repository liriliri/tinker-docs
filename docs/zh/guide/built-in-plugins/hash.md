# tinker-hash

哈希计算插件，支持 MD5、SHA1、SHA256 和 SHA512 算法。

![截图](/hash.png)

## 功能特性

- **多种算法** — 同时计算 MD5、SHA1、SHA256 和 SHA512 哈希值
- **文本模式** — 实时计算文本输入的哈希值
- **文件模式** — 计算任意格式文件的哈希值
- **大写切换** — 在小写和大写哈希输出之间切换
- **剪贴板支持** — 从剪贴板粘贴文本或文件，点击复制单个哈希结果
- **Web Worker** — 文件哈希计算在后台线程中运行，提升性能

## 使用方法

### 文本模式

1. 在工具栏中选择**文本**模式
2. 在输入面板中输入文本
3. 右侧面板会即时显示四种算法的哈希结果
4. 点击任意哈希值旁的复制按钮即可复制到剪贴板

### 文件模式

1. 在工具栏中选择**文件**模式
2. 点击或拖拽文件到输入区域
3. 四种算法的哈希结果将被计算并显示
4. 点击任意哈希值旁的复制按钮即可复制到剪贴板
