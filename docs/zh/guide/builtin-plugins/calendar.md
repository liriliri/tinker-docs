# tinker-calendar

日历插件，提供完整的日历功能，支持事件管理、多种视图以及中西方节假日和农历显示。

![截图](/calendar.png)

## 功能特性

- **多种视图** — 在日视图、周视图和月视图之间切换
- **事件管理** — 创建、编辑和删除日历事件，支持全天和定时事件
- **拖拽操作** — 通过拖拽将事件移动到新的日期或时间
- **节假日显示** — 显示中西方节假日
- **农历显示** — 在月视图中显示农历日期
- **侧边栏** — 查看所选日期的事件列表
- **双语界面** — 支持中文和英文

## 使用方法

1. 使用工具栏在日视图、周视图和月视图之间切换
2. 点击添加按钮或双击日期来创建新事件
3. 填写事件标题、起止日期和时间，可选标记为全天事件
4. 点击已有事件进行编辑或删除
5. 切换侧边栏查看所选日期的所有事件
6. 使用前进/后退按钮或点击"今天"在日期之间导航
