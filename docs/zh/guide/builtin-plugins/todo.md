# tinker-todo

[TINKER](https://github.com/liriliri/tinker) 的待办清单插件，使用 [todo.txt](http://todotxt.org/) 格式进行纯文本任务管理。

![Screenshot](/todo.png)

## 功能特性

- **Todo.txt 格式** — 使用标准 todo.txt 纯文本格式存储任务，便于移植
- **打开或创建文件** — 打开已有的 todo.txt 文件或创建新文件，支持最近文件列表
- **添加任务** — 快速添加新任务，可设置优先级和截止日期
- **优先级** — 为任务设置高 (A)、中 (B)、低 (C) 或无优先级
- **截止日期** — 为任务设置截止日期，支持按时间筛选
- **筛选视图** — 按全部、今天、重要（优先级 A）或已完成筛选任务
- **搜索** — 按文本搜索任务，快速查找项目
- **编辑任务** — 内联编辑任务文本、优先级和截止日期
- **完成/取消完成** — 切换任务完成状态，自动记录完成日期
- **显示/隐藏已完成** — 切换已完成任务的可见性
- **清除已完成** — 一次性删除所有已完成的任务
- **重新加载与在文件夹中显示** — 从磁盘重新加载文件或在文件管理器中显示
- **状态持久化** — 自动保存筛选条件、显示已完成偏好和文件路径

## 使用方法

1. 首次启动时，点击**打开文件**打开已有的 todo.txt 文件，或点击**创建文件**新建一个
2. 在输入框中输入任务，点击**添加**创建任务
3. 添加或编辑任务时可设置**优先级**（高/中/低）和可选的**截止日期**
4. 使用侧边栏筛选器（**全部**、**今天**、**重要**、**已完成**）聚焦特定任务
5. 点击任务的复选框将其标记为已完成
6. 点击任务的**编辑**按钮修改文本、优先级或截止日期
7. 使用搜索栏按关键词查找任务
8. 点击**清除已完成**删除所有已完成的任务
