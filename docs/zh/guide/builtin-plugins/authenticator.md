# tinker-authenticator

两步验证插件，支持 TOTP 验证码生成、密码保护和二维码扫描。

![截图](/authenticator.png)

## 功能特性

- **TOTP 验证码生成** — 支持 SHA1、SHA256、SHA512 算法，6 位或 8 位验证码
- **密码保护** — 使用 AES-GCM 加密账户密钥，支持锁定/解锁
- **二维码支持** — 从截图或图片扫描二维码，也可显示二维码用于分享
- **导入账户** — 通过 `otpauth://` URI 导入
- **搜索** — 按发行方或账户名快速筛选
- **自动刷新** — 验证码在到期时自动刷新，带倒计时指示

## 使用方法

1. 点击 **+** 按钮手动添加账户、扫描二维码或通过 URI 导入
2. TOTP 验证码会自动生成和刷新
3. 点击验证码即可复制到剪贴板
4. 可设置密码来加密账户密钥
