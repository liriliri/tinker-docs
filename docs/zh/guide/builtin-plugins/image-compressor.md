# tinker-image-compressor

图片压缩插件，支持批量压缩 PNG、JPG 和 WebP 格式图片。

![截图](/image_compressor.png)

## 功能特性

- **批量压缩** — 一次压缩多张图片
- **格式支持** — 支持 PNG、JPG 和 WebP 格式
- **可调质量** — 质量滑块范围 1 到 100，提供预设级别（极低、低、中等、高、极佳）
- **压缩前后对比** — 原图与压缩图的可视化并排对比
- **EXIF 保留** — 可选在压缩过程中保留 EXIF 元数据
- **覆盖或另存** — 选择覆盖原文件或将压缩图片保存到新目录
- **大小统计** — 查看每张图片的原始大小、压缩大小和压缩比
- **剪贴板支持** — 将压缩后的图片复制到剪贴板

## 使用方法

1. 点击**打开**或将图片拖拽到插件中添加图片
2. 调整**质量**滑块设置所需的压缩级别
3. 可选开启**保留 EXIF** 以保留图片元数据
4. 点击**压缩**压缩所有图片
5. 点击任意图片的对比按钮查看压缩前后对比
6. 点击**保存**保存所有压缩图片——选择覆盖原文件或保存到新目录
