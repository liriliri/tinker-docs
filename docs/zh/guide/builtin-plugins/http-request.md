# tinker-http-request

HTTP 请求测试插件，类似 Postman，用于发送和检查 HTTP 请求。

![截图](/http_request.png)

## 功能特性

- **多种 HTTP 方法** — 支持 GET、POST、PUT、PATCH、DELETE、HEAD 和 OPTIONS
- **请求参数** — 通过键值编辑器编辑查询参数，自动与 URL 同步
- **自定义请求头** — 添加和管理请求头
- **请求体** — 支持 JSON、URL 编码表单数据和纯文本等请求体类型
- **身份认证** — 内置 Basic Auth 和 Bearer Token 支持
- **响应查看** — 以文本或十六进制模式查看响应体，显示状态码、耗时和大小
- **响应头** — 以表格形式查看响应头
- **集合管理** — 将请求组织到集合和文件夹中，方便管理
- **自动保存** — 切换请求时自动保存集合中的请求更改

## 使用方法

1. 从下拉菜单中选择 HTTP 方法（GET、POST 等）
2. 在地址栏中输入请求 URL
3. 可选在请求面板的标签页中配置参数、请求头、请求体或身份认证
4. 点击**发送**按钮或按 **Enter** 键发送请求
5. 在响应面板中查看响应体、状态码、耗时和响应头

### 集合管理

1. 点击侧边栏中的**新建集合**创建集合
2. 右键点击集合或文件夹可添加请求或子文件夹
3. 点击已保存的请求即可加载到编辑器中
4. 切换请求时更改会自动保存
