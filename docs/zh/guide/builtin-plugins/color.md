# tinker-color

颜色工具插件，提供交互式颜色选择器，支持多种颜色格式实时转换和配色方案。

![截图](/color.png)

## 功能特性

- **色轮选择器** — 交互式色轮，直观选择颜色
- **取色器工具** — 从屏幕上直接拾取任意颜色
- **多种颜色格式** — 实时转换 HEX、RGB、HSL、HSV、CMYK、HSI 和 CIE-LAB 格式
- **Alpha 通道支持** — 通过专用滑块调整透明度
- **饱和度与亮度滑块** — 通过可视化渐变滑块微调颜色属性
- **配色方案** — 查看互补色、类似色和亮度变体
- **复制到剪贴板** — 快速复制显示格式或 CSS 兼容格式的颜色值
- **粘贴颜色** — 粘贴任意支持的颜色字符串进行导入
- **状态持久化** — 选中的颜色和设置跨会话保留

## 使用方法

1. 使用色轮选择基础颜色，或点击取色器图标从屏幕上拾取颜色
2. 使用色轮下方的滑块调整饱和度、亮度和透明度
3. 在七种支持的格式（HEX、RGB、HSL、HSV、CMYK、HSI、LAB）中查看和编辑颜色值
4. 点击任意格式旁的复制按钮，将颜色值复制到剪贴板
5. 在底部浏览配色方案：颜色变体、互补色和类似色
6. 点击配色方案中的任意色块，将其设为当前颜色
