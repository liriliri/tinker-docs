# tinker-text-diff

[TINKER](https://github.com/liriliri/tinker) 的文本对比工具，提供并排编辑和差异查看功能，支持语法高亮。

![Screenshot](/text_diff.png)

## 功能特性

- **编辑模式** — 在并排的双编辑器中输入原始文本和修改后的文本
- **对比模式** — 以高亮方式查看两段文本之间的新增和删除内容
- **语法高亮** — 选择编程语言，在两个编辑器中启用语法高亮
- **语言自动检测** — 打开文件时自动检测编程语言
- **打开文件** — 从文件加载文本到原始或修改编辑器
- **交换文本** — 一键交换原始文本和修改文本
- **剪贴板支持** — 从剪贴板粘贴文本到任一编辑器，或将内容复制到剪贴板
- **撤销/重做** — 编辑器支持完整的撤销和重做操作
- **差异统计** — 在对比模式下查看新增和删除的数量
- **状态持久化** — 自动保存并恢复文本内容、视图模式和语言选择

## 使用方法

1. 切换到**编辑模式**，在**原始**和**修改**面板中输入或粘贴文本
2. 可选择点击**打开文件**从文件加载文本
3. 从语言选择器中选择编程语言以启用语法高亮
4. 切换到**对比模式**查看两段文本的高亮差异对比
5. 使用**交换**按钮交换原始文本和修改文本
6. 点击**清除**重置两个编辑器
