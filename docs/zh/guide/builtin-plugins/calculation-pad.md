# tinker-calculation-pad

计算稿纸插件，提供记事本风格的计算器，支持逐行输入数学表达式并实时显示计算结果。

![截图](/calculation_pad.png)

## 功能特性

- **逐行计算** — 以记事本的形式逐行输入数学表达式
- **实时求值** — 输入时即时计算并显示结果
- **点击复制** — 点击任意结果即可复制到剪贴板
- **自动填充结果** — 按回车键跳到下一行，若下一行为空则自动填入上一行的结果
- **数字格式化** — 结果以千位分隔符显示，提升可读性
- **双语界面** — 支持中文和英文

## 使用方法

1. 在任意一行输入数学表达式（例如 `1 + 2 * 3`）
2. 结果会显示在该行右侧
3. 点击结果即可复制到剪贴板
4. 按回车键跳到下一行
5. 使用工具栏按钮清除当前行、复制结果或清除所有行
