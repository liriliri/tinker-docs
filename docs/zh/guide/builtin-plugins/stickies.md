# tinker-stickies

[TINKER](https://github.com/liriliri/tinker) 的便签插件，支持创建、编辑和管理多彩便签，并提供富文本编辑功能。

![Screenshot](/stickies.png)

## 功能特性

- **富文本编辑** — 支持粗体、斜体、下划线、删除线和文本高亮等格式
- **颜色自定义** — 提供六种背景颜色可选：黄色、绿色、蓝色、粉色、紫色和橙色
- **文本高亮** — 支持多种高亮颜色，包括绿色、蓝色、粉色、紫色和橙色
- **搜索** — 通过内容快速搜索和筛选便签
- **弹出窗口** — 将便签在独立的置顶窗口中打开，方便随时查看
- **持久化存储** — 使用 IndexedDB 自动保存便签数据
- **右键菜单格式化** — 选中文本后右键即可快速应用格式

## 使用方法

1. 启动插件，以网格布局查看所有便签
2. 点击 **+** 按钮创建新便签
3. 点击便签进入编辑模式，开始输入内容
4. 选中文本后右键可应用粗体、斜体、高亮等格式
5. 将鼠标悬停在颜色圆圈图标上，可更改便签的背景颜色
6. 点击**在新窗口打开**按钮，将便签弹出到独立的置顶窗口中
7. 使用搜索栏按内容筛选便签
8. 点击**删除**按钮并确认，即可移除便签
