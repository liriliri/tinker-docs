# tinker-downloader

下载管理器插件，支持暂停、续传和进度跟踪。

![截图](/downloader.png)

## 功能特性

- **添加下载任务** — 输入 URL 和文件名开始下载，自动检测剪贴板中的 URL
- **暂停与续传** — 暂停正在进行的下载，稍后继续
- **进度跟踪** — 实时进度条、已下载/总大小和下载速度显示
- **自定义保存目录** — 选择文件保存位置，设置跨会话保留
- **筛选视图** — 在下载中和已完成标签页之间切换管理任务
- **在文件夹中显示** — 在系统文件管理器中打开已完成的下载
- **复制 URL** — 快速复制下载链接到剪贴板
- **清除已完成** — 一键移除所有已完成和已取消的下载
- **任务持久化** — 重新打开插件时恢复下载任务

## 使用方法

1. 点击工具栏中的 **+ 新建任务** 打开添加下载对话框
2. 输入下载 URL（如果剪贴板中有 URL，会自动填入）
3. 设置文件名并选择保存目录
4. 点击**下载**开始下载
5. 使用**下载中** / **已完成**标签页筛选任务列表
6. 点击任务上的暂停/继续按钮控制下载
7. 点击已完成任务上的文件夹图标在文件管理器中打开
