# tinker-timer

[TINKER](https://github.com/liriliri/tinker) 的秒表和倒计时插件，采用翻页时钟显示风格。

![Screenshot](/timer.png)

## 功能特性

- **秒表** — 精确的秒表功能，显示分钟、秒和百分之一秒
- **分段计时** — 在秒表运行时记录分段时间，显示每段的分段时间和累计时间
- **最快/最慢分段** — 当有两个或更多分段时，自动高亮显示最快和最慢的分段
- **倒计时** — 设置小时、分钟和秒进行倒计时
- **声音提醒** — 倒计时归零时播放提示音
- **开始/暂停/重置** — 秒表和倒计时均支持完整的控制操作
- **状态持久化** — 自动保存并恢复倒计时时长和当前选项卡

## 使用方法

1. 在**秒表**和**倒计时**选项卡之间切换
2. **秒表**：点击**开始**计时，**分段**记录分段时间，**暂停**暂停计时，**重置**清除记录
3. **倒计时**：设置所需的小时、分钟和秒，然后点击**开始**开始倒计时
4. 倒计时归零时会播放声音提醒
