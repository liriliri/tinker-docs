# tinker-hex-editor

HEX 编辑插件，可在字节级别查看和编辑二进制文件。

![截图](/hex_editor.png)

## 功能特性

- **十六进制视图** — 以十六进制格式显示文件内容，提供交互式编辑器
- **字节编辑** — 直接在十六进制视图中修改单个字节
- **打开文件** — 通过文件对话框或拖拽方式打开文件
- **保存与另存为** — 将更改保存回原文件或保存到新位置
- **未保存更改检测** — 跟踪修改状态，仅在有更改时启用保存按钮
- **大文件支持** — 加载大文件时显示加载指示器

## 使用方法

1. 点击工具栏中的**打开**按钮，或将文件拖拽到窗口中
2. 文件内容将以十六进制格式显示
3. 点击任意字节值即可直接编辑
4. 点击**保存**将更改写回文件，或使用**另存为**保存到新位置
