# tinker-timestamp

[TINKER](https://github.com/liriliri/tinker) 的时间戳转换插件，支持时间戳与日期之间的双向转换，覆盖多个时区。

![Screenshot](/timestamp.png)

## 功能特性

- **当前时间戳** — 实时显示当前 Unix 时间戳，每秒更新
- **日期转时间戳** — 将选定的日期和时间转换为 Unix 时间戳
- **时间戳转日期** — 将 Unix 时间戳转换为可读的日期和时间
- **毫秒/秒单位** — 在毫秒和秒精度之间切换时间戳单位
- **时区支持** — 支持全球 25 个时区（UTC-11:00 至 UTC+12:00），附带城市标签
- **复制到剪贴板** — 一键复制转换结果到剪贴板
- **状态持久化** — 自动保存并恢复所选单位和时区

## 使用方法

1. 在顶部查看实时显示的**当前时间戳**
2. 选择时间戳单位：**毫秒**或**秒**
3. 从下拉菜单中选择**时区**
4. 日期转时间戳：在**日期 → 时间戳**区域选择日期和时间
5. 时间戳转日期：在**时间戳 → 日期**区域输入时间戳
6. 点击**复制**将结果复制到剪贴板
