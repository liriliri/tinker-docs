# tinker-renamer

[TINKER](https://github.com/liriliri/tinker) 的批量文件重命名工具，支持多种重命名规则并可实时预览。

![Screenshot](/renamer.png)

## 功能特性

- **批量文件重命名** — 一次重命名多个文件，实时预览新文件名
- **添加文件或文件夹** — 可添加单个文件或整个文件夹中的所有文件
- **替换规则** — 查找并替换文件名中的文本，支持正则表达式、区分大小写和全部匹配选项
- **插入规则** — 为文件名添加前缀或后缀
- **删除规则** — 删除文件名中匹配的文本，支持正则表达式和区分大小写选项
- **格式化规则** — 更改文件名大小写格式：UPPERCASE、lowercase、Capitalize Each Word、kebab-case、snake_case、camelCase 或首字母大写
- **模板规则** — 使用 JavaScript 模板表达式，支持 `${name}`、`${ext}`、`${index}` 等变量进行高级重命名
- **规则管理** — 添加、编辑、排序、启用/禁用和删除规则；规则按顺序应用并自动保存
- **包含扩展名选项** — 可选择替换规则是否同时作用于文件扩展名

## 使用方法

1. 点击**添加文件**或**添加文件夹**将文件添加到列表
2. 点击**添加规则**创建重命名规则（替换、插入、删除、格式化或模板）
3. 配置规则选项后点击**保存**
4. **新名称**列会实时预览重命名后的文件名
5. 根据需要添加更多规则——规则按从上到下的顺序依次应用
6. 可随时切换规则的启用/禁用状态、调整顺序或编辑规则
7. 点击**执行**进行重命名操作
