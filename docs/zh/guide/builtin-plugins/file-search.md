# tinker-file-search

[TINKER](https://github.com/liriliri/tinker) 的快速本地文件搜索插件，可通过文件名快速查找计算机上的文件。

![Screenshot](/file_search.png)

## 功能特性

- **即时搜索** — 输入关键字后实时显示搜索结果，支持防抖输入
- **文件详情** — 在表格视图中显示文件名称、路径、大小和修改时间
- **文件图标** — 显示每种文件类型的系统原生图标
- **文件预览** — 切换侧边面板预览选中的文件
- **无限滚动** — 滚动到底部时自动加载更多结果
- **在文件夹中显示** — 点击文件路径或使用右键菜单在系统文件管理器中定位文件
- **复制路径** — 快速复制文件的完整路径到剪贴板
- **文件删除** — 直接从搜索结果中删除文件，可选择移到回收站或永久删除
- **右键菜单** — 右键点击任意结果可访问在文件夹中显示、复制路径和删除操作

## 使用方法

1. 在顶部搜索栏输入文件名或关键字
2. 在表格中浏览搜索结果，点击某行即可选中
3. 使用工具栏中的按钮切换预览面板查看文件预览
4. 双击搜索结果可在系统文件管理器中打开其所在位置
5. 右键点击搜索结果可使用复制路径或删除等功能
6. 删除时可选择将文件移到回收站或永久删除
