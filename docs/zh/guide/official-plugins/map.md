# tinker-map

地图插件，提供交互式地图界面，支持地点浏览和搜索。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-map/screenshot.png)

## 功能特性

- **交互式地图** — 基于 Leaflet 的地图浏览
- **地点搜索** — 按名称或描述搜索地点
- **地点列表** — 侧边栏展示所有地点，点击可在地图上定位
- **深色模式** — 支持深色主题
- **国际化** — 支持中英文

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-map
```

## 使用方法

1. 在地图上或侧边栏列表中浏览地点
2. 使用关键词搜索地点
3. 点击地点在地图上聚焦查看
