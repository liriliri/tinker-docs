# tinker-js13k

[TINKER](https://github.com/liriliri/tinker) 的 JS13K 游戏插件，精选 JS13K 竞赛作品，可直接在应用内畅玩。

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-js13k/screenshot.png)

## 功能

- **16 款精选游戏**，来自 JS13K 竞赛（2017–2025）
- **搜索**，按名称、作者或描述搜索游戏
- **响应式网格**布局，自动适配窗口大小
- **应用内游玩**，支持全屏模式
- **深色模式**支持

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-js13k
```

## 使用

1. 在网格视图中浏览游戏合集
2. 使用搜索栏搜索游戏
3. 点击游戏卡片开始游玩
4. 点击返回按钮回到游戏列表
