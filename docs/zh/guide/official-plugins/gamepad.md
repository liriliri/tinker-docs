# tinker-gamepad

手柄测试插件，支持多个控制器的实时输入可视化。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-gamepad/screenshot.png)

## 功能特性

- **实时输入**可视化，包括按钮、摇杆和扳机
- **多手柄支持**，可切换玩家
- **摇杆轨迹追踪**，记录摇杆移动历史
- **深色模式**支持

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-gamepad
```

## 使用方法

1. 通过 USB 或蓝牙连接手柄，按下任意按钮
2. 查看实时按钮状态、摇杆数值和控制器图示
3. 使用顶部的 P1/P2/P3/P4 按钮切换玩家
4. 点击 **CLEAR PATHS** 清除摇杆移动轨迹
