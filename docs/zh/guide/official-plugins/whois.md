# tinker-whois

WHOIS 查询插件，支持域名、IP 地址和 ASN 查询。

![截图](https://raw.githubusercontent.com/liriliri/tinker-whois/master/screenshot.png)

## 功能特性

- **域名、IP、ASN** 查询
- **快速响应**（直接 WHOIS TCP 连接）
- **现代化界面**

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-whois
```

## 使用方法

1. 在输入框中输入域名、IP 地址或 ASN
2. 点击"查询"按钮或按回车键
3. 查看 WHOIS 查询结果

## 查询示例

- `google.com` - 查询 Google 域名信息
- `github.com` - 查询 GitHub 域名信息
- `8.8.8.8` - 查询 Google DNS 服务器信息
- `AS15169` - 查询 Google 的 ASN 信息
