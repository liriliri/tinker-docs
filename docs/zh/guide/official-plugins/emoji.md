# tinker-emoji

Emoji 表情选择器插件，支持 1700+ 个表情符号，可按分类浏览和搜索。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-emoji/screenshot.png)

## 功能特性

- **1700+ 表情符号**，按分类浏览
- **搜索功能**，支持按名称、描述或关键词搜索（双语）
- **点击复制**到剪贴板
- **深色模式**支持

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-emoji
```

## 使用方法

1. 按分类浏览表情符号
2. 使用关键词搜索表情符号
3. 点击即可复制表情符号到剪贴板
