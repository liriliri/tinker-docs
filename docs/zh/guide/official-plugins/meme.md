# tinker-meme

表情包搜索插件，支持表情包搜索、浏览和复制到剪贴板。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-meme/screenshot.png)

## 功能特性

- **关键词搜索**表情包
- 搜索框为空时展示**热门表情包**
- **无限滚动**加载更多结果
- **点击复制**图片到剪贴板
- **右键保存**图片到本地
- **深色模式**支持
- **国际化**支持中英文

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-meme
```

## 使用方法

1. 在搜索框输入关键词并按回车
2. 在网格中浏览表情包；向下滚动加载更多
3. 点击表情包即可复制到剪贴板
4. 右键表情包选择"保存"即可保存到本地

## 数据来源

表情包来源于[搜狗图片](https://pic.sogou.com/)。
