# tinker-wallpaper

壁纸插件，搜索和浏览高清壁纸，支持预览、下载和设置为桌面壁纸。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-wallpaper/screenshot.png)

## 功能特性

- **壁纸搜索** — 关键词搜索高清壁纸
- **壁纸浏览** — 网格展示壁纸，无限滚动加载更多
- **壁纸预览** — 查看壁纸大图，支持多种分辨率选择
- **设置壁纸** — 一键设置为桌面壁纸，自动匹配屏幕最佳分辨率
- **保存壁纸** — 下载壁纸到本地
- **双语界面** — 支持中文和英文
- **深色模式** — 支持深色主题

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-wallpaper
```

## 使用方法

1. 在搜索栏输入关键词搜索壁纸
2. 浏览壁纸网格，滚动加载更多
3. 点击壁纸打开预览
4. 选择合适的分辨率
5. 点击设置按钮将壁纸设为桌面背景，或点击保存按钮下载到本地
