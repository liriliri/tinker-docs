# tinker-bilibili-downloader

[TINKER](https://github.com/liriliri/tinker) 的 B 站视频下载插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-bilibili-downloader/screenshot.png)

## 功能特性

- **视频下载**：粘贴 B 站视频链接即可下载
- **画质选择**：支持 1080p、720p、480p、360p 等多种画质
- **多分P支持**：支持番剧和多分P视频
- **下载进度**：任务列表实时跟踪下载进度
- **SESSDATA**：支持 Cookie 登录以获取更高画质

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-bilibili-downloader
```

## 使用方法

1. 粘贴 B 站视频链接，点击**解析**
2. 选择所需画质和分P（多分P视频）
3. 点击**下载**开始
4. 在任务列表中查看下载进度
