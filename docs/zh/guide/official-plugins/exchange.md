# tinker-exchange

汇率换算插件，支持 150+ 种货币的实时汇率计算。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-exchange/screenshot.png)

## 功能特性

- **150+ 种货币**，实时汇率
- **搜索**，通过货币代码或名称查找
- **自定义货币列表**，追踪常用货币
- **深色模式**支持

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-exchange
```

## 使用方法

1. 输入金额并选择基准货币
2. 查看列表中所有货币的换算结果
3. 通过搜索对话框添加或移除货币
