# tinker-typing-test

打字测速插件，实时追踪 WPM/CPM 和准确率统计。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-typing-test/screenshot.png)

## 功能特性

- **WPM 和 CPM** — 实时速度追踪
- **准确率** — 错误高亮监控
- **双语支持** — 支持英文和中文文章
- **60 秒测试** — 带可视化倒计时的计时测试
- **深色模式** — 支持深色主题

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-typing-test
```

## 使用方法

1. 使用语言切换按钮选择语言
2. 开始输入显示的文本即可开始测试
3. 60 秒后查看 WPM、CPM 和准确率结果
4. 点击**再练一次**使用新文章重新测试
