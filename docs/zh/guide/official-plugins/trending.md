# tinker-trending

热搜榜插件，聚合多个平台的热门话题和热搜排行，一目了然。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-trending/screenshot.png)

## 功能特性

- **多平台聚合** — 支持 Hacker News、GitHub、知乎、微博、V2EX、哔哩哔哩、豆瓣、掘金、今日头条、少数派、贴吧、Steam
- **自定义数据源** — 自由添加或移除展示的平台
- **拖拽排序** — 拖拽卡片调整平台的显示顺序
- **一键刷新** — 批量刷新所有平台数据
- **双语界面** — 支持中文和英文
- **深色模式** — 支持深色主题

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-trending
```

## 使用方法

1. 打开插件后自动加载各平台热搜数据
2. 点击条目可跳转到原始链接
3. 点击 **+** 按钮添加或移除平台
4. 拖拽卡片调整显示顺序
5. 点击刷新按钮更新所有平台数据
