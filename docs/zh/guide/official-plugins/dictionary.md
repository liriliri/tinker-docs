# tinker-dictionary

词典插件，支持 MDX 和 ZIP 格式词典文件查词。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-dictionary/screenshot.png)

## 功能特性

- **MDX 与 ZIP 支持** — 加载 MDX 或 ZIP 格式的词典文件
- **多词典管理** — 同时管理和搜索多个词典
- **前缀搜索** — 输入时实时显示自动补全建议
- **丰富释义** — 渲染包含 CSS 和图片的词典释义
- **双语界面** — 支持中文和英文
- **深色模式** — 支持深色主题

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-dictionary
```

## 使用方法

1. 点击**添加词典**加载 MDX 或 ZIP 格式的词典文件
2. 在搜索栏输入单词，查看自动补全建议
3. 选择单词查看释义
4. 在侧边栏切换词典，或选择**全部**跨所有词典搜索
5. 右键点击词典可以移除或在文件管理器中显示
