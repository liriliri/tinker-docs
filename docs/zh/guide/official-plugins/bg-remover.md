# tinker-bg-remover

基于 [@imgly/background-removal-node](https://github.com/nicojones/background-removal-js) 的 [TINKER](https://github.com/liriliri/tinker) 背景去除插件。

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-bg-remover/screenshot.png)

## 功能特性

- **本地处理** — 无需云端依赖，所有处理均在本地完成
- **模型选择** — 可选择 Small 或 Medium 模型大小
- **拖拽加载** — 通过拖拽或文件选择器加载图片
- **PNG 导出** — 将透明结果保存为 PNG 格式
- **国际化** — 支持中英文界面

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-bg-remover
```

## 使用方法

1. 通过拖拽或文件夹按钮打开图片
2. 选择模型大小（Small 或 Medium）
3. 点击**去除背景**
4. 将结果保存为 PNG
