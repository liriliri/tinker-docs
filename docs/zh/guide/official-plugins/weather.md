# tinker-weather

天气插件，提供实时天气信息和天气预报。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-weather/screenshot.png)

## 功能特性

- **实时天气** — 查看温度、湿度、风力和紫外线指数
- **3 天预报** — 温度范围条显示未来天气
- **城市搜索** — 支持拼音搜索城市
- **最近城市** — 快速切换最近查看的城市
- **深色模式** — 支持深色主题

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-weather
```

## 使用方法

1. 在侧边栏搜索城市
2. 查看当前天气和 3 天预报
3. 在最近访问的城市之间切换
