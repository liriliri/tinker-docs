# tinker-translate

翻译插件，支持 Google 翻译、Bing 翻译和 AI 翻译，提供多语言互译功能。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-translate/screenshot.png)

## 功能特性

- **多翻译引擎** — 支持 Google 翻译、Bing 翻译和 AI 翻译
- **语言自动检测** — 自动识别源文本语言
- **语言互换** — 一键交换源语言和目标语言
- **粘贴翻译** — 快速粘贴剪贴板内容进行翻译
- **复制结果** — 一键复制翻译结果
- **双语界面** — 支持中文和英文
- **深色模式** — 支持深色主题

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-translate
```

## 使用方法

1. 从顶部下拉菜单选择翻译引擎（Google、Bing 或 AI）
2. 选择源语言和目标语言，或使用自动检测
3. 在左侧输入框中输入或粘贴需要翻译的文本
4. 点击 **翻译** 按钮开始翻译
5. 在右侧面板查看翻译结果
6. 点击复制按钮复制翻译结果
