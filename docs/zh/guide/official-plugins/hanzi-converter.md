# tinker-hanzi-converter

汉字转换插件，支持拼音转换、人民币大写转换和简繁体转换。

![截图](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-hanzi-converter/screenshot.png)

## 功能特性

- **拼音转换** — 将汉字转换为拼音，支持声调、数字声调和无声调三种模式
- **人民币大写** — 将数字金额转换为人民币大写格式
- **简繁转换** — 支持简体转繁体和繁体转简体
- **一键复制** — 快速复制转换结果到剪贴板
- **双语界面** — 支持中文和英文

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-hanzi-converter
```

## 使用方法

1. 选择所需的转换工具（拼音、人民币大写、简繁转换）
2. 在输入框中输入需要转换的文本
3. 转换结果会实时显示在下方
4. 点击复制按钮即可复制结果
