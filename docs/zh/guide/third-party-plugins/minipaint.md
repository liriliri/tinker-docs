# tinker-minipaint

[TINKER](https://github.com/liriliri/tinker) 的 miniPaint 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-minipaint/screenshot.png)

## 关于

该插件将 [miniPaint](https://github.com/viliusle/miniPaint) 集成到 TINKER 中，这是一个基于 HTML5 技术的在线图形编辑工具，可以创建和编辑图像。miniPaint 提供了一套全面的功能，包括图层、特效和各种绘图工具，非常适合直接在浏览器中进行图像编辑和平面设计。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-minipaint
```
