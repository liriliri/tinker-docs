# tinker-jsoncrack

[TINKER](https://github.com/liriliri/tinker) 的 JSON Crack 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-jsoncrack/screenshot.png)

## 关于

该插件将 [JSON Crack](https://github.com/AykutSarac/jsoncrack.com) 集成到 TINKER 中，这是一个强大的 JSON 可视化工具。JSON Crack 将 JSON 数据转换为交互式图表，使理解复杂数据结构、调试 API 和以可视化树形或图形方式浏览 JSON 文件变得更加容易。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-jsoncrack
```
