# tinker-ctool

[TINKER](https://github.com/liriliri/tinker) 的 Ctool 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-ctool/screenshot.png)

## 关于

该插件将 [Ctool](https://github.com/baiy/Ctool) 集成到 TINKER 中，这是一个面向开发者的综合工具箱。Ctool 提供了丰富的常用开发工具，包括编码/解码、加密/解密、格式转换以及许多其他开发者工具。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-ctool
```
