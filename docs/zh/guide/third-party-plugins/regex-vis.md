# tinker-regex-vis

[TINKER](https://github.com/liriliri/tinker) 的 Regex Vis 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-regex-vis/screenshot.png)

## 关于

该插件将 [Regex-Vis](https://github.com/Bowen7/regex-vis) 集成到 TINKER 中，这是一个正则表达式可视化和编辑工具。Regex-Vis 提供了直观的可视化界面，用于理解和测试正则表达式，具有实时模式可视化和交互式编辑功能。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-regex-vis
```
