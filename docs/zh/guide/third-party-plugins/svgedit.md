# tinker-svgedit

[TINKER](https://github.com/liriliri/tinker) 的 SVGEdit 插件。

## 关于

该插件将 [SVG-Edit](https://github.com/SVG-Edit/svgedit) 集成到 TINKER 中，这是一个强大的基于网页的 SVG 编辑器。SVG-Edit 提供了完整的矢量图形编辑体验，支持绘制形状、路径、文本以及渐变、图案和滤镜等高级 SVG 功能。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-svgedit
```
