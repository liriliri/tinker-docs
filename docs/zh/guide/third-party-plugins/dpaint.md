# tinker-dpaint

[TINKER](https://github.com/liriliri/tinker) 的 DPaint 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-dpaint/screenshot.png)

## 关于

该插件将 [DPaint](https://github.com/steffest/DPaint-js) 集成到 TINKER 中，这是经典 Deluxe Paint 程序的重制版。DPaint 是一个传奇的位图图形编辑器，最初由 Electronic Arts 为 Amiga 计算机开发，现已为现代网页浏览器重新打造。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-dpaint
```
