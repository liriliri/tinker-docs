# tinker-excalidraw

[TINKER](https://github.com/liriliri/tinker) 的 Excalidraw 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-excalidraw/screenshot.png)

## 关于

该插件将 [Excalidraw](https://github.com/excalidraw/excalidraw) 集成到 TINKER 中，这是一个用于绘制手绘风格图表的虚拟协作白板。Excalidraw 提供了直观的绘图体验，支持形状、箭头、文本和实时协作功能。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-excalidraw
```
