# tinker-hoppscotch

[TINKER](https://github.com/liriliri/tinker) 的 Hoppscotch 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-hoppscotch/screenshot.png)

## 关于

该插件将 [Hoppscotch](https://github.com/hoppscotch/hoppscotch) 集成到 TINKER 中，这是一个开源的 API 开发生态系统。Hoppscotch 提供了强大的界面，用于测试和调试 REST、GraphQL、WebSocket 和其他 API 协议，具有请求集合、环境变量、认证辅助和实时响应可视化等功能。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-hoppscotch
```
