# tinker-jspaint

[TINKER](https://github.com/liriliri/tinker) 的 JSPaint 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-jspaint/screenshot.png)

## 关于

该插件将 [JS Paint](https://github.com/1j01/jspaint) 集成到 TINKER 中，这是一个像素级精确的网页版 MS Paint 重制版。JS Paint 忠实地重现了经典的 Windows 画图体验，并带有无限撤销、透明图像编辑和触屏支持等现代改进。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-jspaint
```
