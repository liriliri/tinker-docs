# tinker-online3dviewer

[TINKER](https://github.com/liriliri/tinker) 的 Online 3D Viewer 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-online3dviewer/screenshot.png)

## 关于

该插件集成了 [Online 3D Viewer](https://github.com/kovacsv/Online3DViewer)，一个免费开源的在线 3D 模型查看器，支持多种文件格式，包括 OBJ、STL、GLTF、GLB、3DM、OFF、PLY、BIM 等。

## 安装

下载安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-online3dviewer
```
