# tinker-blockbench

[TINKER](https://github.com/liriliri/tinker) 的 Blockbench 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-blockbench/screenshot.png)

## 关于

该插件集成了 [Blockbench](https://github.com/JannisX11/blockbench)，一个免费的现代化低多边形和方块模型编辑器，支持像素艺术纹理。Blockbench 主要用于 Minecraft 和实体建模，但也支持通用的 3D 建模和动画，提供纹理绘制、动画制作和多种格式导出功能。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-blockbench
```
