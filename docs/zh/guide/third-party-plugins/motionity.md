# tinker-motionity

[TINKER](https://github.com/liriliri/tinker) 的 Motionity 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-motionity/screenshot.png)

## 关于

该插件集成了 [Motionity](https://github.com/alyssaxuu/motionity)，一个免费开源的网页端动态图形编辑器。Motionity 将 After Effects 的强大功能与 Canva 的易用性相结合，提供关键帧、遮罩、图像和视频滤镜、文字动画、音频支持和 Lottie 支持等功能。无需注册即可免费创建动画视频。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-motionity
```
