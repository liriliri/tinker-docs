# tinker-audiomass

[TINKER](https://github.com/liriliri/tinker) 的 AudioMass 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-audiomass/screenshot.png)

## 关于

该插件集成了 [AudioMass](https://github.com/pkalogiros/AudioMass)，一个免费的全功能网页端音频和波形编辑工具。AudioMass 提供了一套完整的音频编辑功能，包括剪切、裁剪、特效、滤镜和格式转换，非常适合直接在浏览器中进行音频编辑任务。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-audiomass
```
