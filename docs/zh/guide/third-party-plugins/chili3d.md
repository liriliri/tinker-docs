# tinker-chili3d

[TINKER](https://github.com/liriliri/tinker) 的 Chili3D 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-chili3d/screenshot.png)

## 关于

该插件集成了 [Chili3D](https://github.com/xiangechen/chili3d)，一个基于网页的 3D CAD 应用，用于在线模型设计和编辑。Chili3D 通过将 OpenCascade (OCCT) 编译为 WebAssembly 并与 Three.js 集成，实现了接近原生的性能，可以直接在浏览器中进行强大的在线建模、编辑和渲染。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-chili3d
```
