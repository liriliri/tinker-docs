# tinker-drawio

[TINKER](https://github.com/liriliri/tinker) 的 Draw.io 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-drawio/screenshot.png)

## 关于

该插件将 [draw.io](https://github.com/jgraph/drawio) 集成到 TINKER 中，这是一个可配置的图表/白板可视化应用。Draw.io 提供了强大的绘图界面，支持流程图、UML 图、网络图以及许多其他类型的技术绘图。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-drawio
```
