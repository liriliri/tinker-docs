# tinker-klecks

[TINKER](https://github.com/liriliri/tinker) 的 Klecks 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-klecks/screenshot.png)

## 关于

该插件将 [Klecks](https://github.com/bitbof/klecks) 集成到 TINKER 中，这是一个强大的网页端绘画应用。Klecks 提供了一套全面的绘画工具，包括画笔、图层、滤镜和特效，非常适合直接在浏览器中进行数字艺术创作和图像编辑。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-klecks
```
