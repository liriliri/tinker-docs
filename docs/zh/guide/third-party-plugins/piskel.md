# tinker-piskel

[TINKER](https://github.com/liriliri/tinker) 的 Piskel 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-piskel/screenshot.png)

## 关于

该插件将 [Piskel](https://github.com/piskelapp/piskel) 集成到 TINKER 中，这是一个免费的在线像素艺术编辑器和动画精灵创作工具。Piskel 提供了简单直观的界面，用于创建像素艺术、动画和游戏精灵，具有洋葱皮、图层和帧管理等功能。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-piskel
```
