# tinker-method-draw

[TINKER](https://github.com/liriliri/tinker) 的 Method Draw 插件。

![截图](https://raw.githubusercontent.com/liriliri/tinker-third-party/master/packages/tinker-method-draw/screenshot.png)

## 关于

该插件将 [Method Draw](https://github.com/methodofaction/Method-Draw) 集成到 TINKER 中，这是一个基于网页的矢量绘图应用。Method Draw 提供了简单易用的 SVG 编辑体验，它有意去掉了图层和线帽/线角等功能，以换取更流畅愉悦的绘图体验，非常适合快速创建和编辑矢量图形。

## 安装

下载并安装 [TINKER](https://tinker.liriliri.io/)，然后运行：

```bash
npm i -g tinker-method-draw
```
