# tinker-trap-the-cat

点击圆点围住小猫，别让它跑到边缘！一款为 [TINKER](https://github.com/liriliri/tinker) 设计的六边形网格益智游戏。

![截图](https://raw.githubusercontent.com/liriliri/tinker-games/master/packages/tinker-trap-the-cat/screenshot.png)

## 亮点

- **经典玩法** — 在六边形网格上围住小猫，别让它到达边缘
- **聪明的对手** — 小猫会选择通往边界路径最多的逃跑路线
- **公平开局** — 随机初始墙壁会经过检查，确保谜题始终可解
- **支持撤销** — 需要第二次机会时可以撤回上一步操作
- **令人满意的反馈** — 小猫逃脱时会喵喵叫，获胜时播放胜利音效
- **双语支持** — 完整支持英文和中文（圈小猫）

## 安装

直接在[浏览器](https://tinker.liriliri.io/games/trap-the-cat/)中游玩，或通过 TINKER 安装：从 `https://tinker.liriliri.io/` 下载后，运行 `npm i -g tinker-trap-the-cat`。

## 玩法

1. 在 TINKER 中打开**圈小猫**插件
2. 点击空白的六边形格子放置墙壁，围住小猫
3. 每步操作后，小猫会向最近的边缘移动一格
4. 成功将小猫团团围住即可获胜 — 如果小猫到达边界则失败
5. 使用**重置**重新开始，或使用**撤销**回退上一步操作
